다음 사용 사례는 LodgingReservation 스키마가 사용되는 방식에 관한 일반적인 예를 보여줍니다. 다음 예를 사용하여 마크업이 올바르게 구성되었는지 확인하세요.

마크업 테스트
이메일 마크업 테스터 도구 [https://www.google.com/webmasters/markup-tester/?hl=ko]를 사용하여 마크업의 유효성을 검사할 수 있습니다. 마크업 코드를 붙여넣고 확인 버튼을 클릭하여 콘텐츠를 스캔하면 오류에 관한 보고서를 받게 됩니다.
사양
이메일 세부정보를 검토하여 예약에 적용되는 추가 숙박 시설이 있는지 확인하세요. 이러한 추가 숙박 시설을 마크업하면 Google에서 사용자에게 숙박 시설 예약에 대해 훨씬 풍부한 설명을 표시할 수 있습니다.
LodgingReservation
유형 이름: LodgingReservation [https://developers.google.com/workspace/gmail/markup/reference/types/LodgingReservation?hl=ko]
예약 [https://developers.google.com/workspace/gmail/markup/reference/types/Reservation?hl=ko] 확장
이름 유형 설명
bookingAgent Person [https://developers.google.com/workspace/gmail/markup/reference/types/Person?hl=ko] 또는 Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] 예약 대행사 또는 대행사입니다. 문자열 (예: '')도 허용합니다.
bookingAgent.name 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 에이전트/서비스의 이름입니다.
bookingAgent.url URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 에이전트/서비스의 웹사이트입니다.
bookingTime DateTime [https://developers.google.com/workspace/gmail/markup/reference/types/DateTime?hl=ko] 예약 날짜입니다.
cancelReservationUrl URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 예약을 취소할 수 있는 웹페이지입니다.
checkinDate
(필수) DateTime [https://developers.google.com/workspace/gmail/markup/reference/types/DateTime?hl=ko] 체크인 시간입니다.
checkoutDate
(필수) DateTime [https://developers.google.com/workspace/gmail/markup/reference/types/DateTime?hl=ko] 결제 시간
checkinUrl URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 숙박자가 체크인할 수 있는 웹페이지입니다.
confirmReservationUrl URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 예약을 확인할 수 있는 웹페이지입니다.
lodgingUnitDescription 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 객실 유형에 관한 텍스트 설명입니다 (스위트/룸, 침대 크기 등 포함).
modifiedTime DateTime [https://developers.google.com/workspace/gmail/markup/reference/types/DateTime?hl=ko] (확인 카드/검색 답변에 권장) 예약이 마지막으로 수정된 시간입니다.
modifyReservationUrl URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] (확인 카드/검색 답변에 권장) 예약을 수정할 수 있는 웹페이지입니다.
numAdults 숫자 [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] 숙박 시설에 투숙하는 성인 수입니다.
numChildren 숫자 [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] 숙박 시설에 투숙하는 어린이 수입니다.
price 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 숙박 시설 예약의 총 가격입니다.
priceCurrency 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 숙박 시설 예약 가격의 통화 (3자리 ISO 4217 형식)입니다.
programMembership ProgramMembership [https://developers.google.com/workspace/gmail/markup/reference/types/ProgramMembership?hl=ko] 예약에 적용되는 항공사 멤버십, 호텔 포인트 제도 등의 멤버십입니다.
programMembership.memberNumber 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 멤버십의 식별자입니다.
programMembership.program 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 프로그램의 이름입니다.
reservationFor
(필수) LodgingBusiness [https://developers.google.com/workspace/gmail/markup/reference/types/LodgingBusiness?hl=ko] 예약한 숙박 시설입니다.
reservationFor.address
(필수) PostalAddress [https://developers.google.com/workspace/gmail/markup/reference/types/PostalAddress?hl=ko] 숙박 시설의 주소입니다.
reservationFor.address.addressCountry
(필수) Text [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 또는 Country [https://developers.google.com/workspace/gmail/markup/reference/types/Country?hl=ko] 숙박 시설의 주소 국가입니다.
reservationFor.address.addressLocality
(필수) 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 숙박 시설 주소의 지역 (예: 도시)입니다.
reservationFor.address.addressRegion
(필수) 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 숙박 시설 주소의 지역 (예: 주)입니다.
reservationFor.address.postalCode
(필수) 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 숙박 시설 주소의 우편번호입니다.
reservationFor.address.streetAddress
(필수) 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 숙박 시설 주소의 상세 주소입니다.
reservationFor.image URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 숙박 시설 업체 사진
reservationFor.name
(필수) 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 숙박 시설의 주소입니다.
reservationFor.telephone
(필수) 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] LodgingBusiness의 전화번호입니다.
reservationFor.url URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 숙박업체의 웹사이트입니다.
reservationNumber
(필수) 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 예약의 번호 또는 ID입니다.
reservationStatus
(필수) ReservationStatus [https://developers.google.com/workspace/gmail/markup/reference/types/ReservationStatus?hl=ko] 예약의 현재 상태입니다.
underName
(필수) Person [https://developers.google.com/workspace/gmail/markup/reference/types/Person?hl=ko] 또는 Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] 게스트
underName.email 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 이메일 주소입니다.
underName.name
(필수) 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 사람의 이름입니다.
url URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 예약을 볼 수 있는 웹페이지입니다.
