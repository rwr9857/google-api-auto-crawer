메시지 ID별로 여러 메시지를 삭제합니다. 메시지가 이미 삭제되지 않았거나 전혀 존재하지 않았다는 보장은 제공하지 않습니다.
HTTP 요청
POST https://gmail.googleapis.com/gmail/v1/users/{userId}/messages/batchDelete
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
