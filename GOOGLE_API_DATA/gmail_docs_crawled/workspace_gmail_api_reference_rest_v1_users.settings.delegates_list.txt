지정된 계정의 위임자를 나열합니다.
이 메서드는 도메인 전체 권한이 위임된 서비스 계정 클라이언트에서만 사용할 수 있습니다.
HTTP 요청
GET https://gmail.googleapis.com/gmail/v1/users/{userId}/settings/delegates
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
userId
string
사용자의 이메일 주소 특수 값 'me'를 사용하여 인증된 사용자를 나타낼 수 있습니다.
요청 본문
요청 본문은 비어 있어야 합니다.
응답 본문
delegates.list 메서드에 대한 응답입니다.
성공한 경우 응답 본문은 다음과 같은 구조의 데이터를 포함합니다.
JSON 표현
{
  "delegates": [
    {
      object (Delegate [https://developers.google.com/workspace/gmail/api/reference/rest/v1/users.settings.delegates?hl=ko#Delegate])

delegates[]
object (Delegate [https://developers.google.com/workspace/gmail/api/reference/rest/v1/users.settings.delegates?hl=ko#Delegate])
사용자의 위임받은 사용자 목록 (인증 상태 포함) 계정에 위임받은 사용자가 없는 경우 이 필드가 표시되지 않습니다.
