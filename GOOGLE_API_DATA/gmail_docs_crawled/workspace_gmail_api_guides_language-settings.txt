이 페이지의 내용
표시 언어 [https://developers.google.com/workspace/gmail/api/guides/language-settings?hl=ko#display_language]
설정 [https://developers.google.com/workspace/gmail/api/v1/reference/users/settings?hl=ko]을 사용하여 계정의 언어 설정을 구성할 수 있으며, 이 설정은 Gmail을 표시할 언어를 설정합니다.
언어 설정을 가져오거나 [https://developers.google.com/workspace/gmail/api/v1/reference/users/settings/getLanguage?hl=ko] 업데이트 [https://developers.google.com/workspace/gmail/api/v1/reference/users/settings/updateLanguage?hl=ko]하는 방법에 관한 자세한 내용은 설정 참조 [https://developers.google.com/workspace/gmail/api/v1/reference/users/settings?hl=ko]를 참고하세요.
표시 언어
설정 참조 [https://developers.google.com/workspace/gmail/api/v1/reference/users/settings?hl=ko]에 설명된 대로 get [https://developers.google.com/workspace/gmail/api/v1/reference/users/settings/getLanguage?hl=ko] 및 update [https://developers.google.com/workspace/gmail/api/v1/reference/users/settings/updateLanguage?hl=ko] 언어 설정 메서드의 표시 언어 형식은 RFC 3066 언어 태그 형식 [https://www.w3.org/International/articles/language-tags]입니다.
일부 Gmail 클라이언트에서는 동일한 언어 집합을 표시할 수 없습니다. 특정 클라이언트에서 사용자의 표시 언어를 사용할 수 없는 경우 해당 클라이언트는 지원되는 가장 가까운 변형 (또는 적절한 기본값)으로 표시하도록 자동으로 선택합니다.
Gmail에서 지원하는 언어는 시간이 지남에 따라 달라지므로 언어 설정 도움말 [https://support.google.com/mail/answer/17091?hl=ko]에 설명된 대로 Gmail 설정 [https://mail.google.com/mail/u/0/?hl=ko#settings/general]의 '언어' 드롭다운에서 사용 가능한 모든 옵션을 참고하세요.
