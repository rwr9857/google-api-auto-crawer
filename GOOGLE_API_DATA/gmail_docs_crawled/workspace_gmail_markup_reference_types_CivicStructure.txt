영업시간 길이 [https://developers.google.com/workspace/gmail/markup/reference/types/Duration?hl=ko] 비즈니스의 영업시간입니다. 영업시간은 주 단위로 지정할 수 있으며 시작은 일 단위로 시작하여 이후의 시간으로 지정합니다. 여러 요일을 쉼표(',')로 구분하여 나열할 수 있습니다. 요일 또는 시간 범위는 하이픈 '-'을 사용하여 지정됩니다.
- 일자는 Mo, Tu, We, Th, Fr, Sa, Su의 두 글자 조합을 사용하여 지정됩니다.
- 시간은 24:00 시간을 사용하여 지정됩니다. 예를 들어 오후 3시는 15:00로 지정됩니다.
- <time itemprop="openingHours" datetime="Tu,Th 16:00-20:00">Tuesdays and Thursdays 4-8pm</time>의 예
- 비즈니스가 연중무휴인 경우 <time itemprop="openingHours" datetime="Mo-Su">Monday through Sunday, all day</time>로 지정할 수 있습니다.
