메서드별 할당량 사용량 [https://developers.google.com/workspace/gmail/api/reference/quota?hl=ko#per-method_quota_usage]
Gmail API에는 API 메서드를 호출할 수 있는 비율을 제한하는 사용량 한도가 적용됩니다. 한도는 Gmail 리소스 사용량을 나타내는 추상 측정 단위인 할당량 단위 [https://developers.google.com/workspace/gmail/api/reference/quota?hl=ko#per-method_quota_usage]로 정의됩니다. 프로젝트별 사용량 한도와 사용자별 사용량 한도라는 두 가지 사용량 한도가 동시에 적용됩니다. 다음 표에는 이러한 사용량 한도가 자세히 나와 있습니다.
사용량 한도 유형 한도 초과 이유
프로젝트별 비율 제한 분당 할당량 단위 1,200,000개 rateLimitExceeded [https://developers.google.com/workspace/gmail/api/guides/handle-errors?hl=ko#resolve_a_403_error_rate_limit_exceeded]
사용자별 비율 제한 사용자별 분당 할당량 단위 15,000개 userRateLimitExceeded [https://developers.google.com/workspace/gmail/api/guides/handle-errors?hl=ko#resolve_a_403_error_user_rate_limit_exceeded]
참고: 한도 오류 처리에 대한 자세한 내용은 오류 해결 [https://developers.google.com/workspace/gmail/api/guides/handle-errors?hl=ko]을 참고하세요.
