이메일 마크업 테스터
이 페이지에는 이메일에 구조화된 데이터를 추가하고 정확성을 검증하는 데 사용할 수 있는 도구와 기법이 나와 있습니다.
자가 테스트
자신에게 이메일을 전송하여 마크업이 처음부터 끝까지 제대로 작동하는지 쉽게 테스트할 수 있습니다. 발신자와 수신자가 동일한 계정인 모든 이메일은 등록 요구사항 [https://developers.google.com/workspace/gmail/markup/registering-with-google?hl=ko]을 무시하며 자체 테스트에 사용될 수 있습니다. DKIM 또는 SPF 인증 자체 테스트에는 여전히 필요합니다.
이 기법을 사용하여 마크업을 엔드 투 엔드로 테스트하고 프로덕션에 통합을 출시할 준비가 되면 Google에 등록 [https://developers.google.com/workspace/gmail/markup/registering-with-google?hl=ko]에서 다음 단계를 확인하세요.
구조화된 데이터 마크업 도우미
구조화된 데이터 마크업 도우미 [https://www.google.com/webmasters/markup-helper/?email=true&hl=ko]에서 Event Reservation, Flight Reservation, Lodging Reservation, Restaurant Reservation 데이터 유형이 포함된 이메일을 마크업하는 방법을 볼 수 있습니다.
이메일 마크업 테스터
이메일 마크업 테스터 [https://www.google.com/webmasters/markup-tester/?hl=ko] 도구를 사용하면 마크업을 테스트하고 문서에서 추출한 구조화된 데이터를 확인할 수 있습니다.
이 도구를 사용하려면 아래 입력 텍스트 상자에 마크업을 붙여넣고 Validate 버튼을 클릭하세요. 도구의 출력에는 추출된 구조화된 데이터와 각 항목의 모든 속성이 포함되어 있으며 입력이 잘못된 경우의 구체적인 오류 메시지가 포함됩니다.
