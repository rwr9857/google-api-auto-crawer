유형 이름: PropertyValueSpecification [https://developers.google.com/workspace/gmail/markup/reference/types/PropertyValueSpecification?hl=ko]
Intangible [https://developers.google.com/workspace/gmail/markup/reference/types/Intangible?hl=ko] 확장
이름 유형 설명
기본값 Text [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 또는 Thing [https://developers.google.com/workspace/gmail/markup/reference/types/Thing?hl=ko] 입력의 기본값입니다. 리터럴을 예상하는 속성의 경우 기본값은 리터럴 값이고, 객체를 예상하는 속성의 경우 현재 값 중 하나의 ID 참조입니다.
maxValue Number [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] 일부 특성 또는 속성의 상한 값입니다.
minValue Number [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] 일부 특성 또는 속성의 하위 값입니다.
여러 값 Boolean [https://developers.google.com/workspace/gmail/markup/reference/types/Boolean?hl=ko] 속성에 여러 값이 허용되는지 여부입니다. 기본값은 false입니다.
읽기 전용 값 Boolean [https://developers.google.com/workspace/gmail/markup/reference/types/Boolean?hl=ko] 속성의 변경 가능 여부입니다. 기본값은 false입니다. 값을 갖는 속성에 이 값을 지정하면 HTML 양식의 '숨겨진' 입력과 유사하게 작동합니다.
단계 값 Number [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] stepValue 속성은 PropertyValueSpecification에서 값의 예상 (및 필수)을 나타냅니다.
값최대 길이 Number [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] 리터럴 값의 문자 수에 허용되는 범위를 지정합니다.
valueMinLength Number [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] 리터럴 값의 문자 수에 허용되는 최소 범위를 지정합니다.
값 이름 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] HTML 템플릿 및 양식 인코딩에 사용되는 PropertyValueSpecification의 이름을 HTML의 input@name과 유사한 방식으로 나타냅니다.
값 패턴 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] HTML 사양에 따라 리터럴 값을 테스트하기 위한 정규 표현식을 지정합니다.
value필수 Boolean [https://developers.google.com/workspace/gmail/markup/reference/types/Boolean?hl=ko] 작업을 완료하기 위해 속성을 입력해야 하는지 여부입니다. 기본값은 false입니다.
