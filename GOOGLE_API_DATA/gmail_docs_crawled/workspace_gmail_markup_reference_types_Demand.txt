GTIN13 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 제품 또는 제품이 참조하는 제품의 13자리 국제거래단품식별코드 [https://en.wikipedia.org/wiki/Global_Trade_Item_Number]입니다. 이는 13자리 ISBN 코드 및 EAN UCC-13과 같습니다. 이전의 12자리 UPC 코드는 앞에 0을 추가하여 GTIN-13 코드로 변환할 수 있습니다. 자세한 내용은 GS1 GTIN 요약 [http://www.gs1.org/barcodes/technical/idkeys/gtin]을 참고하세요.
GTIN14 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 제품 또는 제품이 참조하는 제품의 14자리 글로벌 거래 항목 [https://en.wikipedia.org/wiki/Global_Trade_Item_Number] 코드입니다. 자세한 내용은 GS1 GTIN 요약 [http://www.gs1.org/barcodes/technical/idkeys/gtin]을 참고하세요.
GTIN8 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 제품 또는 제품이 참조하는 제품의 8자리 글로벌 거래 항목 [https://en.wikipedia.org/wiki/Global_Trade_Item_Number] 코드입니다. 이 코드는 EAN/UCC-8 또는 8자리 EAN이라고도 합니다. 자세한 내용은 GS1 GTIN 요약 [http://www.gs1.org/barcodes/technical/idkeys/gtin]을 참고하세요.
includeObject TypeAndQuantityNode [https://developers.google.com/workspace/gmail/markup/reference/types/TypeAndQuantityNode?hl=ko] 혜택에 포함된 제품의 정확한 수량을 나타내는 노드로 연결됩니다.

priceSpecification PriceSpecification [https://developers.google.com/workspace/gmail/markup/reference/types/PriceSpecification?hl=ko] 단가 및 배송료 또는 결제 비용을 나타내는 하나 이상의 상세 가격 사양입니다.

일련번호 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 특정 제품의 일련번호 또는 영숫자 식별자입니다. 혜택에 연결된 경우 혜택에 포함된 제품의 일련번호를 바로가기로 지정할 수 있습니다.
SKU 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 재고 관리 단위 (SKU)로, 제품 또는 서비스 또는 제품이 참조하는 제품의 판매자별 식별자입니다.
