Gmail 메일 가져오기 도구 [https://developers.google.com/workspace/gmail/api/samples?hl=ko#mail_importer_for_gmail]
Gmail 메일 가져오기 도구
Gmail용 메일 가져오기는 Thunderbird 메일 보관 파일의 콘텐츠를 Gmail에 업로드하는 Google의 오픈소스 프로젝트로, 메일의 읽음 상태, 신고 상태, 폴더를 최대한 보존합니다. Java로 작성되었으며 Messages.import [https://developers.google.com/workspace/gmail/api/reference/rest/v1/users.messages/import?hl=ko] 엔드포인트를 사용합니다.
GitHub 저장소 [https://github.com/google/mail-importer]를 클론하여 전체 소스 코드를 가져오거나 Gmail 관련 코드 [https://github.com/google/mail-importer/tree/master/src/main/java/to/lean/tools/gmail/importer/gmail]를 탐색합니다.
