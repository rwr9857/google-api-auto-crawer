이름 유형 설명
운영 가능 수량 QuantitativeValue [https://developers.google.com/workspace/gmail/markup/reference/types/QuantitativeValue?hl=ko] 오퍼 또는 가격 사양이 유효한 주문 수량의 측정 간격 및 단위입니다. 이를 통해 특정 화물 요금이 특정 수량에 대해서만 유효함을 지정할 수 있습니다.
대상 거래 볼륨 PriceSpecification [https://developers.google.com/workspace/gmail/markup/reference/types/PriceSpecification?hl=ko] 혜택 또는 가격 사양이 유효한 통화 단위(예: 최소 구매 수량 표시, 특정 주문량 이상의 무료 배송, 신용카드 구매를 특정 최소 한도로 제한)
maxPrice Number [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] 가격이 범위일 경우 최고 가격입니다.
최소 가격 Number [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] 가격이 범위인 경우 최저 가격입니다.
가격 Number [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] 또는 Text [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 예약의 총 가격입니다.
priceCurrency[가격 통화] 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 예약 가격의 통화 (3자리 ISO 4217 형식)입니다.
유효 기간 시작 DateTime [https://developers.google.com/workspace/gmail/markup/reference/types/DateTime?hl=ko] 상품이 유효해지는 날짜입니다.
유효 기간 DateTime [https://developers.google.com/workspace/gmail/markup/reference/types/DateTime?hl=ko] 혜택의 유효 기간, 가격 사양, 영업시간 데이터가 종료되었습니다.
부가세 포함 Boolean [https://developers.google.com/workspace/gmail/markup/reference/types/Boolean?hl=ko] 가격 사양에 해당하는 부가가치세 (VAT)가 포함되는지 여부를 지정합니다.
