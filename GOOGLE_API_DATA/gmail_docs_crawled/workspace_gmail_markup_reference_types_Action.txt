유형 이름: 작업 [https://developers.google.com/workspace/gmail/markup/reference/types/Action?hl=ko]
Event [https://developers.google.com/workspace/gmail/markup/reference/types/Event?hl=ko] 또는 Thing [https://developers.google.com/workspace/gmail/markup/reference/types/Thing?hl=ko]을 확장합니다.
이름 유형 설명
작업 상태 ActionStatusType [https://developers.google.com/workspace/gmail/markup/reference/types/ActionStatusType?hl=ko] 작업의 현재 처리를 나타냅니다.
에이전트 Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] 또는 Person [https://developers.google.com/workspace/gmail/markup/reference/types/Person?hl=ko] 작업의 직접적인 연기자 또는 동인입니다 (애니메이션 또는 무생물). 예: John이 책을 쓴 경우
endTime DateTime [https://developers.google.com/workspace/gmail/markup/reference/types/DateTime?hl=ko] 항목의 endTime입니다. 예약된 이벤트 또는 서비스 (예: Food시설 예약)의 경우 종료 예정 시간입니다. 일정 기간에 걸친 작업의 경우(예: John은 1월부터 12월까지 책을 썼습니다.
이벤트는 시간이 있는 날짜를 설명할 때도 startTime/endTime 대신 startDate/endDate를 사용합니다. 향후 수정 버전에서 이러한 상황이 더 명확하게 해결될 수 있습니다.
오류 사물 [https://developers.google.com/workspace/gmail/markup/reference/types/Thing?hl=ko] 실패한 작업의 경우 실패 원인에 대한 자세한 정보를 확인하세요.
handler HttpActionHandler [https://developers.google.com/workspace/gmail/markup/reference/types/HttpActionHandler?hl=ko] 회신요청 작업에서 지원하는 핸들러입니다.
악기 사물 [https://developers.google.com/workspace/gmail/markup/reference/types/Thing?hl=ko] 상담사가 작업을 수행하는 데 도움이 된 객체입니다. 예를 들어 John은 펜을 사용하여 책을 작성했습니다.
위치 Place [https://developers.google.com/workspace/gmail/markup/reference/types/Place?hl=ko] 또는 postalAddress [https://developers.google.com/workspace/gmail/markup/reference/types/PostalAddress?hl=ko] 이벤트, 조직 또는 작업의 위치입니다.
이름 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 작업에 연결된 UI 요소에서 사용자에게 표시되는 문자열입니다.
객체 사물 [https://developers.google.com/workspace/gmail/markup/reference/types/Thing?hl=ko] 작업 시 객체가 실행되어 상태가 그대로 유지되거나 변경됩니다. 환자의 영향을 받는 시맨틱 역할, 영향을 받는 환자 또는 질환이 있는 사람 (상태를 변경함) 또는 테마 (예: 변경되지 않음) 등입니다. 예: John은 책을 읽습니다.
참여자 Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] 또는 Person [https://developers.google.com/workspace/gmail/markup/reference/types/Person?hl=ko] 작업에 간접적으로 참여한 다른 공동 대리인(예: 존이 스티브와 함께 책을 쓴 책)
결과 사물 [https://developers.google.com/workspace/gmail/markup/reference/types/Thing?hl=ko] 작업 결과입니다. 예: John은 책을 썼습니다.
startTime DateTime [https://developers.google.com/workspace/gmail/markup/reference/types/DateTime?hl=ko] 항목의 startTime 예약된 이벤트 또는 서비스 (예: Food시설 예약)의 경우 시작이 예상되는 시간입니다. 일정 기간에 걸친 작업의 경우(예: John은 1월부터 12월까지 책을 썼습니다.
이벤트는 시간이 있는 날짜를 설명할 때도 startTime/endTime 대신 startDate/endDate를 사용합니다. 향후 수정 버전에서 이러한 상황이 더 명확하게 해결될 수 있습니다.
target 진입점 [https://developers.google.com/workspace/gmail/markup/reference/types/EntryPoint?hl=ko] 작업의 대상 진입점을 나타냅니다.
url URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 작업을 완료하기 위해 가져올 대상 URL입니다.
