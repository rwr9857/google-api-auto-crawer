발신자 별칭을 업데이트합니다. 서명이 제공되면 Gmail은 HTML을 정리한 후 별칭으로 저장합니다.
계정의 기본 주소 이외의 주소는 도메인 전체 권한이 위임된 서비스 계정 클라이언트만 업데이트할 수 있습니다.
HTTP 요청
PUT https://gmail.googleapis.com/gmail/v1/users/{userId}/settings/sendAs/{sendAsEmail}
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
userId
string
사용자의 이메일 주소 특수 값 'me'를 사용하여 인증된 사용자를 나타낼 수 있습니다.
sendAsEmail
string
업데이트할 발신자 별칭입니다.
요청 본문
요청 본문에 SendAs [https://developers.google.com/workspace/gmail/api/reference/rest/v1/users.settings.sendAs?hl=ko#SendAs]의 인스턴스가 포함됩니다.
응답 본문
성공한 경우 응답 본문에 SendAs [https://developers.google.com/workspace/gmail/api/reference/rest/v1/users.settings.sendAs?hl=ko#SendAs]의 인스턴스가 포함됩니다.
승인 범위
다음 OAuth 범위 중 하나가 필요합니다.
https://www.googleapis.com/auth/gmail.settings.basic
https://www.googleapis.com/auth/gmail.settings.sharing
자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
