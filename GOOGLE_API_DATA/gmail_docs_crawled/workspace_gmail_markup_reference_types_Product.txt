GTIN13 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 제품 또는 제품이 참조하는 제품의 GTIN-13 [http://apps.gs1.org/GDD/glossary/Pages/GTIN-13.aspx] 코드입니다. 이는 13자리 ISBN 코드 및 EAN UCC-13과 같습니다. 이전의 12자리 UPC 코드를 앞에 0을 추가하여 GTIN-13 코드로 변환할 수 있습니다. 자세한 내용은 GS1 GTIN 요약 [http://www.gs1.org/barcodes/technical/idkeys/gtin]을 참고하세요.

GTIN8 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 제품 또는 제품이 참조하는 제품의 GTIN-8 [http://apps.gs1.org/GDD/glossary/Pages/GTIN-8.aspx] 코드입니다. 이 코드는 EAN/UCC-8 또는 8자리 EAN이라고도 합니다. 자세한 내용은 GS1 GTIN 요약 [http://www.gs1.org/barcodes/technical/idkeys/gtin]을 참고하세요.
