계정의 언어 설정입니다. 이 설정은 웹 인터페이스의 '언어 설정' [https://support.google.com/mail/answer/17091?hl=ko] 기능에 해당합니다.
JSON 표현
{
  "displayLanguage": string
}
필드
displayLanguage
string
Gmail을 표시할 언어로, RFC 3066 언어 태그 [https://www.w3.org/International/articles/language-tags/] 형식입니다 (예: 영국 영어, 프랑스어, 일본어의 경우 각각 en-GB, fr, ja).
Gmail에서 지원하는 언어는 시간이 지남에 따라 변경되므로 언어 설정 도움말 [https://support.google.com/mail/answer/17091?hl=ko]에 설명된 대로 Gmail 설정 [https://mail.google.com/mail/u/0/?hl=ko#settings/general]의 '언어' 드롭다운에서 사용 가능한 모든 옵션을 확인하세요. 샘플 값 표는 언어 설정 관리 [https://developers.google.com/workspace/gmail/api/guides/language-settings?hl=ko]를 참고하세요.
일부 Gmail 클라이언트는 동일한 언어를 표시할 수 없습니다. 특정 클라이언트에서 사용자의 표시 언어를 사용할 수 없는 경우 해당 클라이언트는 가장 근접한 지원되는 변형 (또는 적절한 기본값)으로 자동 표시합니다.
