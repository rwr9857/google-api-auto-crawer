티켓 구매가 필요한 이벤트 예약
Gmail의 중요한 기능은 하이라이트입니다. 이름에서 알 수 있듯이 이메일의 주요 정보와 작업을 표시하며, 이메일 메시지에 보기 쉬운 칩으로 표시합니다.
예를 들어 사용자가 항공편 예약 확인을 받으면 이메일 메시지 위에 여행 요약과 체크인 링크가 포함된 칩이 표시될 수 있습니다. 주문 영수증을 받으면 다른 칩에 구매 이미지와 예상 배송일이 표시될 수 있습니다.
발신자는 하이라이트를 사용하여 정보와 작업을 강조할 수 있습니다. 구조화된 데이터 마크업을 사용하여 선언합니다. 지원되는 유형은 다음과 같습니다.

버스, 렌터카, 기차 또는 항공편 예약
예약을 확인하는 이메일에 버스, 렌터카, 기차 또는 항공편 예약 마크업을 사용합니다. 교통편 예약의 하이라이트에는 출발 및 도착 정보와 함께 목적지 도시의 아름다운 이미지가 표시됩니다. 사용 가능한 경우 실시간 상태가 표시되어 최신 정보와 실시간 상태 배지로 강조 표시가 개선됩니다.
항공편 예약 하이라이트는 체크인 작업을 비롯한 작업도 지원합니다.
자세한 내용은 예약 마크업 참조 [https://developers.google.com/workspace/gmail/markup/reference?hl=ko#reservations]를 참고하세요.
주문
구매 (디지털 또는 실제 상품) 영수증이 포함된 이메일에서 주문 마크업을 사용합니다. 사용자에게는 이메일 메시지 위에 비용, 예상 배송, 구매한 상품의 이미지가 포함된 칩이 표시됩니다.
주문 하이라이트는 주문 보기 작업을 비롯한 작업도 지원합니다.
자세한 내용은 주문 마크업 참조 [https://developers.google.com/workspace/gmail/markup/reference/order?hl=ko]를 참고하세요.
소포 배달
주문 배송 알림이 포함된 이메일에서 택배 배송 마크업을 사용합니다. 사용자에게는 주문과 동일한 정보가 포함된 칩이 이메일 메시지 위에 표시되며 최신 상태를 보여주는 실시간 상태 배지도 표시됩니다.
택배 배송 하이라이트도 '택배 추적' 작업을 비롯한 작업을 지원합니다.
자세한 내용은 소포 배송 마크업 참조 [https://developers.google.com/workspace/gmail/markup/reference/parcel-delivery?hl=ko]를 참고하세요.
호텔 예약
호텔 예약 확인 이메일에 호텔 예약 마크업을 사용합니다. 호텔 예약의 하이라이트에는 예약 세부정보와 지정한 이미지가 표시됩니다.
자세한 내용은 호텔 예약 마크업 참조 [https://developers.google.com/workspace/gmail/markup/reference/hotel-reservation?hl=ko]를 참고하세요.
음식점 예약
레스토랑 예약 확인 이메일에 레스토랑 예약 마크업을 사용합니다. 음식점 예약의 하이라이트에는 예약 세부정보와 지정한 이미지가 표시됩니다.
자세한 내용은 레스토랑 예약 마크업 참조 [https://developers.google.com/workspace/gmail/markup/reference/restaurant-reservation?hl=ko]를 참고하세요.
티켓 구매가 필요한 이벤트 예약
이벤트 예약 확인 이메일에서 티켓이 있는 이벤트 예약 마크업을 사용합니다. 티켓이 있는 이벤트 예약의 하이라이트에는 예약의 선택된 세부정보와 지정한 이미지가 표시됩니다.
유료 이벤트 하이라이트에서는 티켓 보기 작업을 비롯한 작업도 지원합니다.
자세한 내용은 티켓 이벤트 마크업 참조 [https://developers.google.com/workspace/gmail/markup/reference/event-reservation?hl=ko]를 참고하세요.
