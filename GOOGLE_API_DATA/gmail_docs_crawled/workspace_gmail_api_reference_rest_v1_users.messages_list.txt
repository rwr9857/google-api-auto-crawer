사용자의 편지함에 있는 메일을 나열합니다. 사용 예는 Gmail 메일 목록 표시 [https://developers.google.com/workspace/gmail/api/guides/list-messages?hl=ko]를 참고하세요.
HTTP 요청
GET https://gmail.googleapis.com/gmail/v1/users/{userId}/messages
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
userId
string
사용자의 이메일 주소 특수 값 me를 사용하여 인증된 사용자를 나타낼 수 있습니다.
쿼리 매개변수
매개변수
maxResults
integer (uint32 [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
반환할 최대 메시지 수입니다. 이 필드의 기본값은 100입니다. 이 필드에 허용되는 최댓값은 500입니다.
pageToken
string
목록에서 특정 결과 페이지를 가져오는 페이지 토큰입니다.
q
string
지정된 쿼리와 일치하는 메시지만 반환합니다. Gmail 검색창과 동일한 쿼리 형식을 지원합니다. 예를 들면 "from:someuser@example.com rfc822msgid:<somemsgid@example.com> is:unread"입니다. gmail.metadata 범위를 사용하여 API에 액세스하는 경우 매개변수를 사용할 수 없습니다.
labelIds[]
string
지정된 모든 라벨 ID와 일치하는 라벨이 있는 메시지만 반환합니다. 대화목록의 메시지에는 동일한 대화목록의 다른 메시지에는 없는 라벨이 있을 수 있습니다. 자세한 내용은 메일 및 대화목록의 라벨 관리하기 [https://developers.google.com/workspace/gmail/api/guides/labels?hl=ko#manage_labels_on_messages_threads]를 참고하세요.
includeSpamTrash
boolean
결과에 SPAM 및 TRASH의 메시지를 포함합니다.

필드
messages[]
object (Message [https://developers.google.com/workspace/gmail/api/reference/rest/v1/users.messages?hl=ko#Message])
메시지 목록입니다. 각 메시지 리소스에는 id와 threadId만 포함됩니다. messages.get [https://developers.google.com/workspace/gmail/api/v1/reference/users/messages/get?hl=ko] 메서드를 사용하여 추가 메시지 세부정보를 가져올 수 있습니다.
nextPageToken
string
목록에서 결과의 다음 페이지를 검색하는 토큰입니다.
resultSizeEstimate
integer (uint32 [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
예상 총 결과 수입니다.

https://mail.google.com/
https://www.googleapis.com/auth/gmail.modify
https://www.googleapis.com/auth/gmail.readonly
https://www.googleapis.com/auth/gmail.metadata
