티켓 토큰 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 또는 URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 바코드 이미지가 사이트에서 호스팅되는 경우 입력란 값은 이미지의 URL 또는 바코드나 QR URI(예: 'barcode128:AB34'(ISO-15417 바코드), 'qrCode:AB34'(QR 코드), 'aztecCode:AB34'(Aztec 코드), 'barcodeEAN:12)CA'(CA) 1234)(CA)
총가격 Number [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko], PriceSpecification [https://developers.google.com/workspace/gmail/markup/reference/types/PriceSpecification?hl=ko] 또는 Text [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 관련 세금, 배송비 등 예약 또는 티켓의 총가격입니다.
