마크업 테스트
이메일 마크업 테스터 도구 [https://www.google.com/webmasters/markup-tester/?hl=ko]를 사용하여 마크업의 유효성을 검사할 수 있습니다. 마크업 코드를 붙여넣고 확인 버튼을 클릭하여 콘텐츠를 스캔하면 오류에 관한 보고서를 받게 됩니다.
사양
다음 표에는 이 유형에 사용할 수 있는 모든 속성이 나와 있습니다.
속성 유형 설명
deliveryAddress PostalAddress [https://developers.google.com/workspace/gmail/markup/reference/types/PostalAddress?hl=ko] (필수) 목적지 주소입니다.
deliveryAddress.name 텍스트 PostalAddress의 이름입니다.
deliveryAddress.streetAddress 텍스트 (필수) 상세 주소입니다. 예: 1600 Amphitheatre Pkwy
deliveryAddress.addressLocality 텍스트 (필수) 지역입니다. (예: 마운틴 뷰)
deliveryAddress.addressRegion 텍스트 (필수) 리전입니다. 예: CA.
deliveryAddress.addressCountry 텍스트 또는 국가 [https://developers.google.com/workspace/gmail/markup/reference/types/Country?hl=ko] (필수) 국가입니다. 미국을 예로 들 수 있습니다. 두 자리 ISO 3166-1 alpha-2 국가 코드를 입력해도 됩니다.
deliveryAddress.postalCode 텍스트 (필수) 우편번호입니다. 예: 94043
originAddress PostalAddress [https://developers.google.com/workspace/gmail/markup/reference/types/PostalAddress?hl=ko] 배송업체의 주소입니다.
originAddress.name 텍스트 PostalAddress의 이름입니다.
originAddress.streetAddress 텍스트 상세 주소입니다. 예: 1600 Amphitheatre Pkwy
originAddress.addressLocality 텍스트 지역입니다. (예: 마운틴 뷰)
originAddress.addressRegion 텍스트 지역입니다. 예: CA.
originAddress.addressCountry 텍스트 또는 국가 [https://developers.google.com/workspace/gmail/markup/reference/types/Country?hl=ko] 국가입니다. 미국을 예로 들 수 있습니다. 두 자리 ISO 3166-1 alpha-2 국가 코드를 입력해도 됩니다.
originAddress.postalCode 텍스트 우편번호입니다. 예: 94043
expectedArrivalFrom DateTime [https://developers.google.com/workspace/gmail/markup/reference/types/DateTime?hl=ko] 택배물이 도착할 것으로 예상되는 가장 빠른 날짜입니다.
expectedArrivalUntil DateTime [https://developers.google.com/workspace/gmail/markup/reference/types/DateTime?hl=ko] (필수) 상품이 도착할 것으로 예상되는 가장 늦은 날짜입니다.
이동통신사 Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] (필수) 택배 배송을 담당하는 당사자입니다. 문자열 (예: 'FedEx')도 허용합니다.
carrier.name 텍스트 (필수) 조직의 이름입니다.
carrier.url URL 조직의 URL입니다.
itemShipped 제품 [https://developers.google.com/workspace/gmail/markup/reference/types/Product?hl=ko] 또는 예약 [https://developers.google.com/workspace/gmail/markup/reference/types/Reservation?hl=ko] (필수) 이 배송에 포함된 제품입니다. 수량을 나타내려면 제품을 적절히 반복하거나 이 속성을 사용하지 않고 대신 일부 제품을 포함하여 포함된 제품을 표현하세요. 객체의 배열도 허용합니다.
itemShipped.name 텍스트 (필수) 제품의 이름입니다.
itemShipped.url URL (확인 카드/검색 답변에 권장) 제품의 URL로, 일반적으로 판매자 웹사이트의 제품 방문 페이지입니다.
itemShipped.image URL (확인 카드/검색 답변에 권장) 제품 이미지의 URL로, 일반적으로 판매자의 웹사이트에 있는 이미지입니다.
itemShipped.sku 텍스트 (확인 카드/검색 답변에 권장) 재고 관리 단위 (SKU)란 제품, 서비스 또는 제품이 참조하는 제품의 판매자별 식별자입니다.
itemShipped.description 텍스트 제품에 대한 간단한 설명입니다.
itemShipped.brand 브랜드 [https://developers.google.com/workspace/gmail/markup/reference/types/Brand?hl=ko] 제품과 연결된 브랜드입니다.
itemShipped.brand.name 텍스트 브랜드 이름입니다.
itemShipped.color 텍스트 제품의 색상입니다.
trackingNumber 텍스트 (확인 카드/검색 답변에 권장) 배송업체 운송장 번호입니다.
trackingUrl URL (확인 카드/검색 답변에 권장) 배송을 추적할 수 있는 웹페이지.
hasDeliveryMethod DeliveryMethod [https://developers.google.com/workspace/gmail/markup/reference/types/DeliveryMethod?hl=ko] 전송에 사용되는 방법입니다.
deliveryStatus DeliveryEvent [https://developers.google.com/workspace/gmail/markup/reference/types/DeliveryEvent?hl=ko] 패키지가 여정의 각 구간 (발송에서 최종 배송까지)을 통과할 때 새 항목이 추가되었습니다. 매장 수령, 보관함 등 수령 방식의 배송에 특히 유용합니다.
partOfOrder 항목 [https://developers.google.com/workspace/gmail/markup/reference/types/Order?hl=ko] (필수) 배송 중인 주문의 세부정보입니다. 배송에 포함된 제품에 대한 세부정보를 itemShipped에서 표현할 수 없는 경우 (예: 발송된 수량) 이 주문을 사용하여 제품을 표현하세요.
partOfOrder.orderNumber 텍스트 (필수) 거래의 판매자별 식별자입니다.
partOfOrder.merchant Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] 또는 Person [https://developers.google.com/workspace/gmail/markup/reference/types/Person?hl=ko] (필수) 주문을 받는 당사자 (예: Amazon.com은 여러 판매자의 판매자임) 문자열도 허용합니다 (예: 'Bob Dole').
partOfOrder.merchant.name 텍스트 (필수) 조직의 이름입니다.
partOfOrder.merchant.sameAs URL 판매자의 Freebase URL입니다.
partOfOrder.orderStatus OrderStatus [https://developers.google.com/workspace/gmail/markup/reference/types/OrderStatus?hl=ko] (확인 카드/검색 답변에 권장) 주문의 현재 상태입니다.
