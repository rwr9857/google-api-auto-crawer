이 페이지에서는 프로모션 이메일에 주석을 추가하는 데 사용되는 유형과 관련 속성을 설명합니다. 이러한 유형 및 속성은 프로모션 이메일의 헤더에 JSON-LD [https://developers.google.com/workspace/gmail/markup/reference/formats/json-ld?hl=ko]로 포함됩니다.
자세한 내용은 이메일 마크업 시작하기 [https://developers.google.com/workspace/gmail/markup/getting-started?hl=ko] 가이드를 참고하세요.
조직
Organization 유형을 사용하면 프로모션에 포함할 로고를 지정할 수 있습니다.
이름 유형 설명
logo ImageObject [https://developers.google.com/workspace/gmail/markup/reference/types/ImageObject?hl=ko] 또는 URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 발신자의 아이콘으로 표시되는 로고입니다. HTTPS URL을 사용하는 것이 좋습니다. 로고 URL 권장사항 [https://developers.google.com/workspace/gmail/promotab/best-practices?hl=ko#logo_url]을 참고하세요.
DiscountOffer
DiscountOffer 유형을 사용하면 할인 코드나 만료일과 같은 제품 세부정보를 홍보하는 거래 주석 [https://developers.google.com/workspace/gmail/promotab/overview?hl=ko#deal-annotation]을 만들 수 있습니다.
이름 유형 설명
description 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 선택사항입니다. 특가 배지 아이콘 옆에 표시되는 짧은 텍스트입니다(예: Free shipping 또는 20% off). 특가 배지 권장사항 [https://developers.google.com/workspace/gmail/promotab/best-practices?hl=ko#deal_badge]을 참고하세요.
discountCode 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 선택사항입니다. 혜택의 할인 또는 프로모션 코드(예: 20PROMO)입니다. 할인 코드 권장사항 [https://developers.google.com/workspace/gmail/promotab/best-practices?hl=ko#discount_code]을 참고하세요.
availabilityStarts DateTime [https://developers.google.com/workspace/gmail/markup/reference/datetime-formatting?hl=ko] 선택사항입니다. 혜택이 시작되는 날짜 및 시간입니다. ISO 8601 [https://support.google.com/merchants/answer/7055760?hl=ko] 형식(예: 2023-09-25T18:44:37-07:00)으로 나타냅니다.
availabilityEnds DateTime [https://developers.google.com/workspace/gmail/markup/reference/datetime-formatting?hl=ko] 선택사항입니다. 프로모션의 종료일 및 종료 시간입니다. ISO 8601 [https://support.google.com/merchants/answer/7055760?hl=ko] 형식(예: 2023-10-25T18:44:37-07:00)으로 나타냅니다. 만료일 권장사항 [https://developers.google.com/workspace/gmail/promotab/best-practices?hl=ko#expiration_date]을 참고하세요.
PromotionCard
PromotionCard 유형을 사용하면 프로모션의 시각적 미리보기를 포함할 수 있습니다.
이름 유형 설명
image ImageObject [https://developers.google.com/workspace/gmail/markup/reference/types/ImageObject?hl=ko] 또는 URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko]
필수 항목입니다. PNG 또는 JPEG 형식의 이메일 이미지 미리보기
지원되는 가로세로 비율은 4:5, 1:1, 1.91:1입니다. 이미지가 자동으로 가운데로 자릅니다. 이미지 권장사항 [https://developers.google.com/workspace/gmail/promotab/best-practices?hl=ko#images]을 참고하세요.
제품 캐러셀의 경우 각 PromotionCard에는 고유한 이미지가 있어야 하며 동일한 가로세로 비율을 사용해야 합니다.
url URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 필수 항목입니다. 프로모션의 URL입니다. 사용자가 이메일의 image를 클릭하면 이 URL로 연결됩니다.
headline 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 선택사항입니다. image 아래에 표시되는 프로모션에 대한 1~2줄의 설명입니다.
price 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko]
선택사항입니다. 프로모션 가격입니다. discountValue를 설정하면 price가 할인이 적용되기 전의 프로모션 원래 가격으로 표시됩니다.
priceCurrency 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 선택사항입니다. 가격에 사용된 통화로, 3글자 ISO 4217 [https://en.wikipedia.org/wiki/ISO_4217] 형식(예: USD)입니다. 가격과 함께 표시되는 통화 기호를 결정합니다.
discountValue 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko]
선택사항입니다. 조정된 가격을 표시하기 위해 price에서 뺀 금액입니다. 조정된 가격이 원래 가격 옆에 표시됩니다.
예를 들어 discountValue가 25로 설정되고 price가 100로 설정되고 priceCurrency가 USD로 설정되면 조정된 가격은 $75로 표시됩니다.
position 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko]
선택사항입니다. 캐러셀에서 카드의 위치입니다.
