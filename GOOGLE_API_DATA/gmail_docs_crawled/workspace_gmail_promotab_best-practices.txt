만료일
이메일 주석을 사용하면 이미지, 할인, 만료일, 기타 혜택을 포함하여 이메일에 생동감을 더할 수 있습니다. 고화질 이미지와 관련성이 가장 높은 정보를 사용하면 Gmail 프로모션 탭에서 이메일을 볼 가능성과 클릭할 가능성이 높아집니다.&ast; [https://developers.google.com/workspace/gmail/promotab/best-practices?hl=ko#footnote]
이 섹션에서는 이메일 메시지에 주석을 추가하기 위한 권장사항을 설명합니다.
이미지
텍스트만 사용하지 마세요. 이 공간은 프로모션 이메일의 시각적 구성요소를 강조하기 위해 설계되었습니다. 실적을 개선하려면 제품 또는 라이프스타일 이미지를 사용해 보세요.
잘 자르세요. 이미지는 자동으로 중앙에서 잘립니다. 지원되는 가로세로 비율은 4:5, 1:1, 1.91:1입니다.
이미지를 재사용하지 마세요. 고유한 이미지를 사용하여 프로모션 이메일을 신선하고 흥미롭게 유지하세요. 동일한 이미지를 재사용하는 이메일 메시지는 사용자가 중복으로 간주할 수 있습니다.
이미지 URL에 HTTP가 아닌 HTTPS를 사용합니다.
특가 배지
'20% 할인' 또는 '무료 배송'과 같은 프로모션의 특징을 강조합니다.
구체적으로 작성합니다. 할인율 또는 프로모션 코드와 같은 프로모션 세부정보를 확인합니다.
긴 문장을 사용하지 마세요. 배지 공간이 제목과 경쟁하므로 할인 배지에 4단어 이상 또는 완전한 문장을 사용하지 마세요.
로고 URL
HTTPS (HTTP 아님) URL을 사용합니다. 이 로고 선택은 이메일 미리보기에만 표시됩니다.
Discount code
이메일에 할인 코드가 표시된 경우에만 할인 코드를 사용하세요. 이 공간은 텍스트 앞에 항상 '코드'라고 표시되므로 용도를 변경하지 마세요.
만료일
이 기능을 사용하면 이메일이 번들 상단에 두 번 미리보기로 표시됩니다. 처음 전송될 때 한 번, 혜택이 만료된 후 3일 이내에 한 번 표시됩니다.
만료된 날짜를 주석에 남겨두면 오래된 혜택으로 인식되어 이메일이 번들에 채워지지 않습니다.
시간 및 시간대를 지정합니다 (예: '2018-12-30T23:59:59-0700'). 날짜 형식 지정에 대한 자세한 내용은 날짜 및 시간 형식 지정 [https://support.google.com/merchants/answer/7055760?hl=ko]을 참고하세요.
&ast; 이메일은 여러 요인을 기반으로 자동으로 그룹화됩니다.
