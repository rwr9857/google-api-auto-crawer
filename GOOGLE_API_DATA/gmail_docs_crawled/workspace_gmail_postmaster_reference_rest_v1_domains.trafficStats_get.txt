특정 날짜의 도메인 트래픽 통계를 가져옵니다. 사용자에게 도메인의 TrafficStats에 액세스할 권한이 없는 경우 PERMISSION_DENIED를 반환합니다.
HTTP 요청
GET https://gmailpostmastertools.googleapis.com/v1/{name=domains/*/trafficStats/*}
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

매개변수
name
string
가져올 트래픽 통계의 리소스 이름입니다. 예: domains/mymail.mydomain.com/trafficStats/20160807

요청 본문은 비어 있어야 합니다.
