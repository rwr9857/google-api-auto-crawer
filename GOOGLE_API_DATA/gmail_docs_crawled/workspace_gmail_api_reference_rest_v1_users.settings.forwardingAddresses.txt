필드
forwardingEmail
string
메일을 전달할 수 있는 이메일 주소입니다.
verificationStatus
enum (VerificationStatus [https://developers.google.com/workspace/gmail/api/reference/rest/v1/users.settings.forwardingAddresses?hl=ko#VerificationStatus])
이 주소가 인증되었으며 전달에 사용할 수 있는지 여부를 나타냅니다. 읽기 전용입니다.
VerificationStatus
전달 사용을 위해 이메일 주소의 소유권이 확인되었는지 여부를 나타냅니다.
열거형
verificationStatusUnspecified 지정되지 않은 인증 상태입니다.
accepted 이제 이 주소를 전달에 사용할 수 있습니다.
pending 주소가 소유자의 확인을 기다리는 중입니다.
