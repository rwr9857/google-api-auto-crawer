유형 이름: WebPage [https://developers.google.com/workspace/gmail/markup/reference/types/WebPage?hl=ko]
CreativeWork [https://developers.google.com/workspace/gmail/markup/reference/types/CreativeWork?hl=ko]를 확장합니다.
이름 유형 설명
탐색경로 BreadcrumbList [https://developers.google.com/workspace/gmail/markup/reference/types/BreadcrumbList?hl=ko] 또는 Text [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 사용자가 웹사이트 계층 구조를 이해하고 탐색하는 데 도움이 되는 링크 집합입니다.
최종 검토됨 날짜 [https://developers.google.com/workspace/gmail/markup/reference/types/Date?hl=ko] 이 웹페이지의 콘텐츠가 정확성 또는 완전성을 마지막으로 검토한 날짜입니다.
기본 콘텐츠 페이지 WebPageElement [https://developers.google.com/workspace/gmail/markup/reference/types/WebPageElement?hl=ko] 이 웹페이지 요소가 페이지의 주요 주제인지를 나타냅니다.
기본 이미지 ImageObject [https://developers.google.com/workspace/gmail/markup/reference/types/ImageObject?hl=ko] 페이지의 기본 이미지를 나타냅니다.
관련 링크 URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 이 웹페이지와 관련된 링크입니다(예: 다른 관련 웹페이지 링크).
검토 대상 Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] 또는 Person [https://developers.google.com/workspace/gmail/markup/reference/types/Person?hl=ko] 이 웹페이지의 콘텐츠에서 정확성 또는 완전성을 검토한 사용자 또는 조직
유의사항 링크 URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 페이지에서 더 중요한 URL 중 하나입니다. 일반적으로 가장 많이 클릭하는 비탐색 링크입니다.
중요한 링크 URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 페이지에서 가장 중요한 URL입니다. 일반적으로 가장 많이 클릭하는 비탐색 링크입니다.
전공 전공 [https://developers.google.com/workspace/gmail/markup/reference/types/Specialty?hl=ko] 이 웹페이지의 콘텐츠가 적용되는 도메인 전문 분야 중 하나입니다.
