클라이언트가 등록한 특정 도메인을 가져옵니다. 도메인이 없는 경우 NOT_FOUND를 반환합니다.
HTTP 요청
GET https://gmailpostmastertools.googleapis.com/v1/{name=domains/*}
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
name
string
도메인의 리소스 이름입니다. 형식은 domains/{domain_name}여야 하며 여기서 domain_name은 정규화된 도메인 이름입니다.
요청 본문
요청 본문은 비어 있어야 합니다.
응답 본문
성공한 경우 응답 본문에 Domain [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains?hl=ko#Domain]의 인스턴스가 포함됩니다.
승인 범위
다음 OAuth 범위가 필요합니다.
https://www.googleapis.com/auth/postmaster.readonly
자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
