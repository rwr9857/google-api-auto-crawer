Gmail Postmaster Tools API
bookmark_border
이 페이지의 내용
서비스: gmailpostmastertools.googleapis.com
검색 문서
서비스 엔드포인트
REST 리소스: v1.domains
REST 리소스: v1.domains.trafficStats
Postmaster Tools API는 현재 Gmail Postmaster Tools UI를 통해 제공되는 이메일 트래픽 측정항목 (예: 스팸 신고, 전송 오류 등)에 프로그래매틱 방식으로 액세스할 수 있는 RESTful API입니다.
서비스: gmailpostmastertools.googleapis.com
이 서비스를 호출하려면 Google에서 제공하는 클라이언트 라이브러리 [https://cloud.google.com/apis/docs/client-libraries-explained?hl=ko]를 사용하는 것이 좋습니다. 애플리케이션에서 자체 라이브러리를 사용하여 이 서비스를 호출해야 한다면 API 요청을 실행할 때 다음 정보를 사용해야 합니다.
검색 문서
검색 문서 [https://developers.google.com/discovery/v1/reference/apis?hl=ko]는 머신에서 읽을 수 있는 사양이며 REST API를 설명하고 사용하는 데 참조할 수 있습니다. 또한 클라이언트 라이브러리, IDE 플러그인, 기타 Google API와 상호작용하는 도구를 빌드하는 데 사용됩니다. 하나의 서비스는 여러 개의 검색 문서를 제공할 수 있습니다. 이 서비스는 다음 검색 문서를 제공합니다.
https://gmailpostmastertools.googleapis.com/$discovery/rest?version=v1 [https://gmailpostmastertools.googleapis.com/$discovery/rest?version=v1]
서비스 엔드포인트
서비스 엔드포인트 [https://cloud.google.com/apis/design/glossary?hl=ko#api_service_endpoint]는 API 서비스의 네트워크 주소를 지정하는 기준 URL입니다. 하나의 서비스는 여러 서비스 엔드포인트를 포함할 수 있습니다. 이 서비스에는 다음 서비스 엔드포인트가 포함되고, 그 아래의 모든 URI가 이 서비스 엔드포인트와 관련됩니다.
https://gmailpostmastertools.googleapis.com
REST 리소스: v1.domains [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains?hl=ko]
메서드
get [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains/get?hl=ko] GET /v1/{name=domains/*}
클라이언트가 등록한 특정 도메인을 가져옵니다.
list [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains/list?hl=ko] GET /v1/domains
클라이언트가 등록한 도메인을 나열합니다.
REST 리소스: v1.domains.trafficStats [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko]
메서드
get [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats/get?hl=ko] GET /v1/{name=domains/*/trafficStats/*}
특정 날짜의 도메인 트래픽 통계를 가져옵니다.
list [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats/list?hl=ko] GET /v1/{parent=domains/*}/trafficStats
사용 가능한 모든 날짜의 트래픽 통계를 표시합니다.
