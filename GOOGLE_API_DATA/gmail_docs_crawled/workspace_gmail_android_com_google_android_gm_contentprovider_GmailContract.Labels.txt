이 문자열 값이 라벨의 표준 이름이 됩니다. 표준 이름은 현지화되지 않으며 사용자에게 표시되지 않습니다.
유형: TEXT
상수 값: "canonicalName"

공개 정적 최종 문자열 CONTENT_TYPE

상수 값: "vnd.android.cursor.dir/vnd.com.google.android.gm.label"
공개 정적 최종 문자열 이름
이 문자열 값은 사용자에게 표시되는 라벨 이름입니다. 시스템 라벨의 이름(받은편지함, 보낸편지함, 임시보관함 등)은 현지화됩니다.
유형: TEXT
상수 값: "name"

이 정수 값은 라벨의 포그라운드 텍스트 색상이며 32비트 0xAARRGGBB 형식입니다.
유형: INTEGER
상수 값: "text_color"
공개 정적 최종 문자열 URI
이 문자열 열 값은 android.content.ContentProvider#query()에 대한 후속 호출에서 이 행에 의해 표시되는 단일 라벨에 관한 정보를 쿼리하는 데 사용할 수 있는 URI입니다.
유형: TEXT
상수 값: "labelUri"
공개 메서드
공개 정적 URI getLabelsUri (문자열 계정)
쿼리 시 계정의 라벨 목록을 반환하는 URI를 반환합니다.
Labels API를 사용하려면 앱에서 먼저 유효한 Gmail 계정의 이메일 주소를 찾아 라벨 정보를 쿼리해야 합니다. AccountManager [http://developer.android.com/reference/android/accounts/AccountManager.html?hl=ko]는 이 정보를 반환할 수 있습니다 (예 [https://developers.google.com/workspace/gmail/android?hl=ko]).
