유형 이름: ItemList [https://developers.google.com/workspace/gmail/markup/reference/types/ItemList?hl=ko]
Intangible [https://developers.google.com/workspace/gmail/markup/reference/types/Intangible?hl=ko] 확장
이름 유형 설명
itemList요소 ListItem [https://developers.google.com/workspace/gmail/markup/reference/types/ListItem?hl=ko], Text [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 또는 Thing [https://developers.google.com/workspace/gmail/markup/reference/types/Thing?hl=ko] itemListElement 값의 경우 간단한 문자열 (예: 'Peter', 'Paul', 'Mary'), 기존 항목 또는 ListItem을 사용할 수 있습니다.

텍스트 값은 목록의 요소가 일반 문자열일 때 가장 적합합니다. 기존 항목은 데이터에 있는 기존 항목의 단순하고 정렬되지 않은 목록에 가장 적합합니다. ListItem은 이 목록의 요소에 관한 추가 컨텍스트를 제공하려고 하거나 동일한 항목이 다른 목록의 다른 위치에 있을 수 있는 경우 순서가 지정된 목록과 함께 사용됩니다.

참고: 마크업의 요소 순서는 순서나 요소를 나타내는 데 충분하지 않습니다. 그러한 경우 ListItem을 'position' 속성과 함께 사용합니다.
itemListOrder ItemListOrderType [https://developers.google.com/workspace/gmail/markup/reference/types/ItemListOrderType?hl=ko] 또는 Text [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 순서의 유형입니다 (예: 오름차순, 내림차순, 정렬되지 않음).
항목 수 Number [https://developers.google.com/workspace/gmail/markup/reference/types/Number?hl=ko] ItemList의 항목 수입니다. 일부 설명은 목록의 모든 항목을 전부 설명하지 않을 수 있습니다 (예: 다중 페이지 페이지 나누기).
