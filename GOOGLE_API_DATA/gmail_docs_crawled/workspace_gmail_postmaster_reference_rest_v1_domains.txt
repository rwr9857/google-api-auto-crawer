리소스: 도메인
Postmaster API에 등록된 도메인 리소스입니다.
JSON 표현
{
  "name": string,
  "createTime": string,
  "permission": enum (Permission [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains?hl=ko#Permission])
}
필드
name
string
도메인의 리소스 이름입니다. 도메인 이름은 domains/{domain_name} 형식이며 여기서 domain_name은 정규화된 도메인 이름 (예: mymail.mydomain.com)입니다.
createTime
string (Timestamp [https://protobuf.dev/reference/protobuf/google.protobuf/#timestamp] format)
사용자가 이 도메인을 등록한 타임스탬프입니다. 서버에서 할당됩니다.
permission
enum (Permission [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains?hl=ko#Permission])
이 도메인에 대한 사용자의 권한입니다. 서버에서 할당됩니다.
권한
사용자가 도메인에 대해 가질 수 있는 권한입니다.
열거형
PERMISSION_UNSPECIFIED 기본값이며 명시적으로 사용해서는 안 됩니다.
OWNER 사용자에게 도메인에 대한 읽기 액세스 권한이 있으며 다른 사용자와 액세스 권한을 공유할 수 있습니다.
READER 사용자에게 도메인에 대한 읽기 액세스 권한이 있습니다.
NONE 사용자에게 도메인 정보에 액세스할 권한이 없습니다. 사용자가 도메인 소유권을 확인하지 않았거나 다른 도메인 소유자가 액세스 권한을 부여하지 않았습니다.
