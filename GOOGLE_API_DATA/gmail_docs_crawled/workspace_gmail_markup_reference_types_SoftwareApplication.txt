파일 형식 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 바이너리의 MIME 형식입니다 (예: application/zip).

operatingSystem 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 지원되는 운영체제 (Windows 7, OSX 10.6, Android 1.6).
권한 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 앱을 실행하는 데 필요한 권한(예: 모바일 앱의 경우 완벽한 인터넷 액세스가 필요할 수도 있고 WiFi에서만 실행될 수도 있음).

출시 노트 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 또는 URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 이 버전의 변경사항에 관한 설명입니다.
요구사항 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 또는 URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 애플리케이션의 구성요소 종속 항목 요구사항입니다. 여기에는 애플리케이션 배포 패키지에 포함되어 있지 않지만 애플리케이션을 실행하는 데 필요한 런타임 환경 및 공유 라이브러리가 포함됩니다 (예: DirectX, 자바 또는 .NET 런타임).
스크린샷 ImageObject [https://developers.google.com/workspace/gmail/markup/reference/types/ImageObject?hl=ko] 또는 URL [https://developers.google.com/workspace/gmail/markup/reference/types/URL?hl=ko] 앱의 스크린샷 이미지 링크입니다.
