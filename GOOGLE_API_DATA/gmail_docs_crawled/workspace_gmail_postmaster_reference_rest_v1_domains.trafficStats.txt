특정 날짜와 관련된 이메일 트래픽 통계
JSON 표현
{
  "name": string,
  "userReportedSpamRatio": number,
  "ipReputations": [
    {
      object (IpReputation [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#IpReputation])
    }
  ],
  "domainReputation": enum (ReputationCategory [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#ReputationCategory]),
  "spammyFeedbackLoops": [
    {
      object (FeedbackLoop [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#FeedbackLoop])
    }
  ],
  "spfSuccessRatio": number,
  "dkimSuccessRatio": number,
  "dmarcSuccessRatio": number,
  "outboundEncryptionRatio": number,
  "inboundEncryptionRatio": number,
  "deliveryErrors": [
    {
      object (DeliveryError [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#DeliveryError])
    }
  ],
  "userReportedSpamRatioLowerBound": number,
  "userReportedSpamRatioUpperBound": number
}
필드
name
string
트래픽 통계의 리소스 이름입니다. 트래픽 통계 이름은 domains/{domain}/trafficStats/{date} 형식입니다. 여기서 domain_name은 이 트래픽 통계와 관련된 도메인의 정규화된 도메인 이름(예: mymail.mydomain.com)이고 date는 이 통계가 해당하는 날짜(yyyymmdd 형식)입니다. 예: domains/mymail.mydomain.com/trafficStats/20160807
userReportedSpamRatio
number
사용자가 신고한 스팸과 받은편지함에 전송된 이메일의 비율입니다. 이는 정확하지 않을 수 있습니다. 더 명시적인 정확성 보장을 위해 간격 필드 userReportedSpamRatioLowerBound [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#TrafficStats.FIELDS.user_reported_spam_ratio_lower_bound] 및 userReportedSpamRatioUpperBound [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#TrafficStats.FIELDS.user_reported_spam_ratio_upper_bound]의 설명을 참고하는 것이 좋습니다. 이 측정항목은 DKIM [http://www.dkim.org/]으로 인증된 이메일에만 적용됩니다.
ipReputations[]
object (IpReputation [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#IpReputation])
도메인의 이메일 서버 IP 주소와 관련된 평판 정보입니다. REPUTATION_CATEGORY_UNSPECIFIED [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#ReputationCategory.ENUM_VALUES.REPUTATION_CATEGORY_UNSPECIFIED]를 제외한 각 평판 카테고리에 정확히 하나의 항목이 있습니다.
domainReputation
enum (ReputationCategory [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#ReputationCategory])
도메인의 평판
spammyFeedbackLoops[]
object (FeedbackLoop [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#FeedbackLoop])
개별 스팸 비율이 포함된 스팸성 피드백 루프 식별자 [https://support.google.com/mail/answer/6254652?hl=ko] 이 측정항목은 DKIM [http://www.dkim.org/]으로 인증된 트래픽에만 적용됩니다.
spfSuccessRatio
number
SPF로 인증에 성공한 메일과 SPF [http://www.openspf.org/]로 인증을 시도한 모든 메일의 비율입니다. 스푸핑된 메일은 제외됩니다.
dkimSuccessRatio
number
DKIM으로 인증에 성공한 메일과 DKIM [http://www.dkim.org/]으로 인증을 시도한 모든 메일의 비율입니다. 스푸핑된 메일은 제외됩니다.
dmarcSuccessRatio
number
DMARC [https://dmarc.org/] 정렬 검사를 통과한 메일과 SPF [http://www.openspf.org/] 또는 DKIM [http://www.dkim.org/]으로 인증에 성공한 도메인에서 수신된 모든 메일의 비율입니다.
outboundEncryptionRatio
number
보안 전송 (TLS)을 통해 수신된 발신 메일 (Gmail에서 전송된 메일)의 비율입니다.
inboundEncryptionRatio
number
보안 전송 (TLS)을 통과한 수신 메일 (Gmail로 전송된 메일)과 해당 도메인에서 수신한 모든 메일의 비율입니다. 이 측정항목은 SPF [http://www.openspf.org/] 또는 DKIM [http://www.dkim.org/]을 통과한 트래픽에만 적용됩니다.
deliveryErrors[]
object (DeliveryError [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#DeliveryError])
도메인의 전송 오류입니다. 이 측정항목은 SPF [http://www.openspf.org/] 또는 DKIM [http://www.dkim.org/]을 통과한 트래픽에만 적용됩니다.
userReportedSpamRatioLowerBound
number
사용자가 신고한 스팸 비율에 대한 신뢰 구간의 하한값입니다. 이 필드가 설정되면 userReportedSpamRatio [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#TrafficStats.FIELDS.user_reported_spam_ratio] 값이 이 간격의 중간점으로 설정되므로 부정확합니다. 하지만 실제 비율은 95% 의 확률로 이 하한값과 상응하는 상한값 사이에 있을 수 있습니다. 이 측정항목은 DKIM [http://www.dkim.org/]으로 인증된 이메일에만 적용됩니다.
userReportedSpamRatioUpperBound
number
사용자가 신고한 스팸 비율에 대한 신뢰 구간의 상한입니다. 이 필드가 설정되면 userReportedSpamRatio [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#TrafficStats.FIELDS.user_reported_spam_ratio] 값이 이 간격의 중간점으로 설정되므로 부정확합니다. 하지만 실제 비율은 95% 의 경우 이 상한과 상응하는 하한 사이에 있을 수 있습니다. 이 측정항목은 DKIM [http://www.dkim.org/]으로 인증된 이메일에만 적용됩니다.
IpReputation
특정 평판 카테고리의 IP 집합에 대한 IP 평판 정보입니다.
JSON 표현
{
  "reputation": enum (ReputationCategory [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#ReputationCategory]),
  "ipCount": string,
  "sampleIps": [
    string
  ]
}
필드
reputation
enum (ReputationCategory [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#ReputationCategory])
이 IP 평판이 나타내는 평판 카테고리입니다.
ipCount
string (int64 [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
이 평판 카테고리의 고유 IP 총 개수입니다. 이 측정항목은 SPF [http://www.openspf.org/] 또는 DKIM [http://www.dkim.org/]을 통과한 트래픽에만 적용됩니다.
sampleIps[]
string
이 평판 카테고리의 IP 샘플
ReputationCategory
도메인 또는 IP의 평판입니다. 자세한 내용은 Postmaster Tools 도움말 페이지 [https://support.google.com/mail/answer/6227174?hl=ko]의 대시보드 섹션에 있는 '도메인 및 IP 평판 대시보드'를 참고하세요.
열거형
REPUTATION_CATEGORY_UNSPECIFIED 명시적으로 사용해서는 안 되는 기본값입니다. 평판 정보를 사용할 수 없는 상태를 나타냅니다.
HIGH 매우 낮은 스팸 비율로 정상적인 기록을 보이며 Gmail의 발신자 가이드라인을 준수합니다. 스팸 필터에 의해 표시되는 경우가 거의 없습니다.
MEDIUM 정상적인 메일을 전송하지만 가끔 소량의 스팸을 전송하는 경향이 있는 것으로 알려져 있습니다. 이 항목의 메일은 스팸 수준이 현저하게 증가하는 경우를 제외하면 대부분 전송률이 양호합니다.
LOW 상당한 양의 스팸을 정기적으로 전송하는 것으로 알려져 있으며 이 발신자의 메일은 스팸으로 표시될 가능성이 큽니다.
BAD 매우 많은 양의 스팸을 전송한 기록이 있습니다. 이 발신자에게서 전송된 메일은 거의 항상 SMTP 수준에서 거부되거나 스팸으로 표시됩니다.
FeedbackLoop
피드백 루프 [https://support.google.com/mail/answer/6254652?hl=ko] 식별자 정보

필드
id
string

spamRatio
number
사용자가 식별자로 스팸으로 표시한 메시지 수와 해당 식별자가 있는 받은편지함 메시지의 총 개수의 비율입니다.
DeliveryError

JSON 표현
{
  "errorClass": enum (DeliveryErrorClass [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#DeliveryErrorClass]),
  "errorType": enum (DeliveryErrorType [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#DeliveryErrorType]),
  "errorRatio": number
}
필드
errorClass
enum (DeliveryErrorClass [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#DeliveryErrorClass])
전송 오류 클래스입니다.
errorType
enum (DeliveryErrorType [https://developers.google.com/workspace/gmail/postmaster/reference/rest/v1/domains.trafficStats?hl=ko#DeliveryErrorType])
전송 오류 유형입니다.
errorRatio
number
오류가 발생한 메일의 비율과 인증된 모든 트래픽의 비율입니다.
DeliveryErrorClass
전송 오류 클래스입니다.
열거형
DELIVERY_ERROR_CLASS_UNSPECIFIED 명시적으로 사용해서는 안 되는 기본값입니다.
PERMANENT_ERROR 메시지 전송이 거부되었습니다.
TEMPORARY_ERROR 수신자에게 메시지를 전송하는 데 일시적인 오류가 발생했습니다.
DeliveryErrorType
전송 오류 유형입니다.
열거형
DELIVERY_ERROR_TYPE_UNSPECIFIED 명시적으로 사용해서는 안 되는 기본값입니다.
RATE_LIMIT_EXCEEDED 도메인 또는 IP에서 의심스럽게 높은 비율로 트래픽을 전송하여 일시적으로 트래픽 제한이 적용되었습니다. Gmail에서 트래픽이 정상이라고 충분히 확신하는 경우 제한이 해제됩니다.
SUSPECTED_SPAM 여러 이유로 트래픽이 Gmail에서 스팸으로 의심됩니다.
CONTENT_SPAMMY 콘텐츠 때문에 트래픽이 스팸으로 의심됩니다.
BAD_ATTACHMENT 트래픽이 Gmail에서 지원하지 않는 첨부파일을 포함합니다.
BAD_DMARC_POLICY 발신자 도메인에서 DMARC 거부 정책을 설정했습니다.
LOW_IP_REPUTATION 발신 IP의 IP 평판이 매우 낮습니다.
LOW_DOMAIN_REPUTATION 발신 도메인의 도메인 평판이 매우 낮습니다.
IP_IN_RBL IP가 하나 이상의 공개 실시간 블랙홀 리스트 [http://en.wikipedia.org/wiki/DNSBL]에 등록되어 있습니다. RBL과 협조하여 IP를 목록에서 삭제합니다.
DOMAIN_IN_RBL 도메인이 하나 이상의 공개 실시간 블랙홀 리스트 [http://en.wikipedia.org/wiki/DNSBL]에 등록되어 있습니다. RBL 관리자에게 연락하여 도메인이 목록에서 삭제되도록 합니다.
BAD_PTR_RECORD 발신 IP에 PTR 레코드 [https://support.google.com/domains/answer/3251147?hl=ko#ptr]가 누락되었습니다.
