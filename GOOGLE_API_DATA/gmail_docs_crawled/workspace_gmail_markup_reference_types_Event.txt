이벤트 상태 이벤트 상태 유형 [https://developers.google.com/workspace/gmail/markup/reference/types/EventStatusType?hl=ko] 이벤트의 eventStatus는 상태를 나타내며 이벤트가 취소되거나 일정이 조정된 경우 특히 유용합니다.
위치 Place [https://developers.google.com/workspace/gmail/markup/reference/types/Place?hl=ko] 또는 postalAddress [https://developers.google.com/workspace/gmail/markup/reference/types/PostalAddress?hl=ko] 이벤트, 조직 또는 작업의 위치입니다.
쿠폰 혜택 [https://developers.google.com/workspace/gmail/markup/reference/types/Offer?hl=ko] 이 항목을 제공하겠다는 제안입니다(예: 제품 판매, 영화 DVD 대여, 이벤트 티켓 증정).
주최자 Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] 또는 Person [https://developers.google.com/workspace/gmail/markup/reference/types/Person?hl=ko] 이벤트 주최자입니다.
연주자 Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] 또는 Person [https://developers.google.com/workspace/gmail/markup/reference/types/Person?hl=ko] 이벤트의 공연자(예: 발표자, 음악가, 음악 그룹 또는 배우)
공연자 Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] 또는 Person [https://developers.google.com/workspace/gmail/markup/reference/types/Person?hl=ko] 이벤트의 주요 공연자(예: 발표자, 뮤지션, 배우)
이전 시작일 날짜 [https://developers.google.com/workspace/gmail/markup/reference/types/Date?hl=ko] 일정이 변경되거나 취소된 이벤트에서 eventStatus와 함께 사용됩니다. 이 속성에는 이전에 예약된 시작일이 포함됩니다. 일정이 조정된 이벤트인 경우 startDate 속성을 새로 예약된 시작 날짜에 사용해야 합니다. (드물지만) 이벤트가 여러 번 연기 및 일정이 조정된 경우 이 필드는 반복될 수 있습니다.
녹화됨 CreativeWork [https://developers.google.com/workspace/gmail/markup/reference/types/CreativeWork?hl=ko] 이 이벤트의 전체 또는 일부를 캡처한 CreativeWork입니다.
startDate 날짜 [https://developers.google.com/workspace/gmail/markup/reference/types/Date?hl=ko] 상품의 시작 날짜 및 시간입니다 (ISO 8601 날짜 형식 [http://ko.wikipedia.org/wiki/ISO_8601]).
하위 이벤트 이벤트 [https://developers.google.com/workspace/gmail/markup/reference/types/Event?hl=ko] 이 이벤트에 속하는 이벤트입니다. 예를 들어 회의 이벤트에는 여러 프레젠테이션이 포함되며, 각 프레젠테이션은 회의의 하위 이벤트입니다.
하위 이벤트 이벤트 [https://developers.google.com/workspace/gmail/markup/reference/types/Event?hl=ko] 이 이벤트에 포함된 일정입니다. 예를 들어 회의 이벤트에는 많은 프레젠테이션이 포함되며 각 하위 이벤트의 회의입니다.
슈퍼 이벤트 이벤트 [https://developers.google.com/workspace/gmail/markup/reference/types/Event?hl=ko] 이 이벤트가 속한 이벤트입니다. 예를 들어 개별 음악 공연 컬렉션의 각 음악은 superEvent로서 음악 축제를 가질 수 있습니다.
일반적인 연령대 텍스트 [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 일반적인 예상 연령대(예: '7~9', '11~')입니다.
작업 수행 CreativeWork [https://developers.google.com/workspace/gmail/markup/reference/types/CreativeWork?hl=ko] 극장 이벤트에서 공연한 공연 등 특정 이벤트에 공연된 작품.
