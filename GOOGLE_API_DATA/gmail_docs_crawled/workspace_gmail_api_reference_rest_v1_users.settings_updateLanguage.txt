언어 설정을 업데이트합니다.
성공하면 반환 객체에 사용자에게 저장된 displayLanguage가 포함되며, 이 값은 요청에 전달된 값과 다를 수 있습니다. 요청된 displayLanguage가 Gmail에서 직접 지원되지는 않지만 유사한 변형이 지원되므로 이 변형이 대신 선택되어 저장될 수 있기 때문입니다.
HTTP 요청
PUT https://gmail.googleapis.com/gmail/v1/users/{userId}/settings/language
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
userId
string
사용자의 이메일 주소 특수 값 'me'를 사용하여 인증된 사용자를 나타낼 수 있습니다.
요청 본문
요청 본문에 LanguageSettings [https://developers.google.com/workspace/gmail/api/reference/rest/v1/LanguageSettings?hl=ko]의 인스턴스가 포함됩니다.
응답 본문
성공한 경우 응답 본문에 LanguageSettings [https://developers.google.com/workspace/gmail/api/reference/rest/v1/LanguageSettings?hl=ko]의 인스턴스가 포함됩니다.
승인 범위
다음 OAuth 범위가 필요합니다.
https://www.googleapis.com/auth/gmail.settings.basic
자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
