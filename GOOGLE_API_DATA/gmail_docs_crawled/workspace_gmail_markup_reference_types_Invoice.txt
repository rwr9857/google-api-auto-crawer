중개인 Organization [https://developers.google.com/workspace/gmail/markup/reference/types/Organization?hl=ko] 또는 Person [https://developers.google.com/workspace/gmail/markup/reference/types/Person?hl=ko] 구매자와 판매자 간에 교환을 주선하는 법인입니다. 대부분의 경우 중개인은 교환과 관련된 제품이나 서비스의 소유권을 획득하거나 해제하지 않습니다. 법인이 중개자, 판매자 또는 구매자인지 불분명한 경우에는 처음 두 용어가 더 좋습니다.
category PhysicalActivityCategory [https://developers.google.com/workspace/gmail/markup/reference/types/PhysicalActivityCategory?hl=ko], Text [https://developers.google.com/workspace/gmail/markup/reference/types/Text?hl=ko] 또는 Thing [https://developers.google.com/workspace/gmail/markup/reference/types/Thing?hl=ko] 상품의 카테고리 큰 기호나 슬래시를 사용하여 카테고리 계층 구조를 비공식적으로 표시할 수 있습니다.
