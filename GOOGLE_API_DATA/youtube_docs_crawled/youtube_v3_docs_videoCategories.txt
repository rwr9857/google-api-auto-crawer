속성
다음 표는 이 리소스에 표시되는 속성을 정의합니다.
속성
kind string
API 리소스의 유형을 식별합니다. 값은 youtube#videoCategory입니다.
etag etag
이 리소스의 ETag입니다.
id string
YouTube에서 동영상 카테고리를 고유하게 식별하는 데 사용하는 ID입니다.
snippet object
snippet 객체에는 제목을 비롯한 동영상 카테고리에 관한 기본 세부정보가 포함되어 있습니다.
snippet.channelId string
동영상 카테고리를 만든 YouTube 채널입니다.
snippet.title string
동영상 카테고리의 제목입니다.
snippet.assignable boolean
동영상을 카테고리와 연결할 수 있는지 여부를 나타냅니다.
