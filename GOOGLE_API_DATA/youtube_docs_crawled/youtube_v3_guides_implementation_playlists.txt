현재 사용자의 재생목록 가져오기
playlists.list [https://developers.google.com/youtube/v3/docs/playlists/list?hl=ko] 메서드를 호출하여 현재 인증된 사용자의 재생목록을 가져옵니다. 요청에서 mine [https://developers.google.com/youtube/v3/docs/playlists/list?hl=ko#mine] 매개변수의 값을 true로 설정합니다. mine 매개변수를 사용하는 요청은 OAuth 2.0을 사용하여 승인해야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.playlists.list?
        part=snippet,contentDetails
        &mine=true [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.playlists.list?part=snippet%252CcontentDetails&mine=true]
사용자의 재생목록 가져오기
이 예에서는 특정 채널에서 만든 재생목록을 검색합니다. 이 예에는 두 단계가 있습니다.
1단계: 채널 ID 가져오기
channels.list [https://developers.google.com/youtube/v3/docs/channels/list?hl=ko] 메서드를 호출하여 특정 YouTube 사용자 이름과 연결된 채널 ID를 가져옵니다. 이 예에서는 사용자 이름이 GoogleDevelopers입니다. API 응답에서 id 속성은 채널 ID를 식별합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.channels.list?
        part=id
        &forUsername=GoogleDevelopers [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.channels.list?part=id&forUsername=GoogleDevelopers]
2단계: 채널의 재생목록 검색
playlists.list [https://developers.google.com/youtube/v3/docs/playlists/list?hl=ko] 메서드를 호출하여 채널의 재생목록을 가져옵니다. channelId [https://developers.google.com/youtube/v3/docs/playlists/list?hl=ko#channelId] 매개변수의 값을 1단계에서 가져온 값으로 설정합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.playlists.list?
        part=snippet,contentDetails
        &channelId=UC_x5XG1OV2P6uZZ5FSM9Ttw [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.playlists.list?part=snippet%252CcontentDetails&channelId=UC_x5XG1OV2P6uZZ5FSM9Ttw]
특정 재생목록에 대한 정보 검색
하나 이상의 특정 재생목록에 대한 정보를 검색하려면 playlists.list [https://developers.google.com/youtube/v3/docs/playlists/list?hl=ko] 메서드를 호출하고 id [https://developers.google.com/youtube/v3/docs/playlists/list?hl=ko#id] 매개변수를 원하는 리소스를 식별하는 YouTube 재생목록 ID의 쉼표로 구분된 목록으로 설정합니다. 이 기능의 일반적인 사용 사례는 검색 결과 집합에서 반환된 재생목록 그룹에 관한 추가 정보를 검색하는 요청입니다. 예를 들어 각 재생목록의 항목 수를 가져올 수 있습니다.
아래 요청은 playlists.list 메서드를 호출하여 검색어 'GoogleDevelopers'와 일치하는 두 재생목록의 항목 수를 검색합니다. 이 정보는 playlist 리소스의 contentDetails.itemCount 속성에 있으므로 요청은 part 매개변수 값을 contentDetails로 설정합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.playlists.list?
        part=contentDetails
        &id=PLOU2XLYxmsIIM9h1Ybw2DuRw6o2fkNMeR,PLyYlLs02rgBYRWBzYpoHz7m2SE8mEZ68w [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.playlists.list?part=contentDetails&id=PLOU2XLYxmsIIM9h1Ybw2DuRw6o2fkNMeR,PLyYlLs02rgBYRWBzYpoHz7m2SE8mEZ68w]
재생목록 추가
이 예에서는 playlists.insert [https://developers.google.com/youtube/v3/docs/playlists/insert?hl=ko] 메서드를 호출하여 인증된 사용자의 채널에 비공개 재생목록을 만듭니다. 이 메서드에 대한 모든 API 요청은 OAuth 2.0을 사용하여 승인되어야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.playlists.insert?
        part=snippet,status [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.playlists.insert?part=snippet%252Cstatus&resource=%257B%250A++%2522snippet%2522%253A+%250A++%257B%250A++++%2522title%2522%253A+%2522New+playlist%2522%252C%250A++++%2522description%2522%253A+%2522New+playlist+description%2522%250A++%257D%252C%250A++%2522status%2522%253A+%250A++%257B%250A++++%2522privacyStatus%2522%253A+%2522private%2522%250A++%257D%250A%257D]
API 탐색기 링크는 API 요청의 본문을 지정하는 매개변수도 설정합니다. 요청 본문에는 재생목록의 제목, 설명, 공개 상태와 같은 정보를 정의하는 playlist [https://developers.google.com/youtube/v3/docs/playlists?hl=ko] 리소스가 포함되어 있습니다. 이 예에서 요청 본문은 다음과 같습니다.
{
  "snippet": {
    "title": "New playlist",
    "description": "New playlist description"
  },
  "status": {
    "privacyStatus": "private"
  }
}
다음 예에서는 새 재생목록을 팟캐스트 프로그램으로 표시합니다.
{
  "snippet": {
    "title": "New playlist",
    "description": "New playlist description"
  },
  "status": {
    "podcastStatus": "enabled"
  }
}

재생목록 업데이트
이 예시에서는 재생목록의 공개 범위 상태를 private에서 public로 업데이트합니다. 이 예에는 두 단계가 있습니다.
1단계: 재생목록 ID 가져오기
playlists.list [https://developers.google.com/youtube/v3/docs/playlists/list?hl=ko] 메서드를 호출하여 현재 인증된 사용자의 채널에 있는 재생목록을 가져옵니다. 위의 현재 사용자의 재생목록을 가져오는 [https://developers.google.com/youtube/v3/guides/implementation/playlists?hl=ko#playlists-retrieve-for-current-user] 샘플 요청은 이 요청을 보여줍니다. API를 호출하는 애플리케이션은 API 응답을 처리하여 각 재생목록의 ID를 키로 사용하여 재생목록 목록을 표시할 수 있습니다.
2단계: 재생목록 업데이트하기
playlists.update [https://developers.google.com/youtube/v3/docs/playlists/update?hl=ko] 메서드를 호출하여 특정 재생목록을 수정합니다. 이 메서드에는 유효한 OAuth 2.0 승인 토큰이 필요합니다.
이 메서드를 호출할 때 snippet.title 속성이 필요하므로 요청 본문에 playlist 리소스의 snippet 부분이 포함되어야 합니다. 또한 업데이트되는 재생목록 리소스에서 snippet.description 또는 snippet.tags 속성의 값을 지정하는 경우 이러한 값을 업데이트 요청에서 다시 지정해야 합니다. 그러지 않으면 값이 삭제됩니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.playlists.update?
        part=snippet,status [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.playlists.update?part=snippet%252Cstatus&resource=%257B%250A++%2522id%2522%253A+%2522PLAYLIST_ID%2522%252C%250A++%2522snippet%2522%253A+%250A++%257B%250A++++%2522title%2522%253A+%2522New+playlist%2522%252C%250A++++%2522description%2522%253A+%2522New+playlist+description%2522%250A++%257D%252C%250A++%2522status%2522%253A+%250A++%257B%250A++++%2522privacyStatus%2522%253A+%2522public%2522%250A++%257D%250A%257D]
위의 재생목록 추가 예에서와 같이 API 탐색기 링크는 API 요청의 본문을 지정하는 매개변수를 설정합니다. 요청 본문에는 재생목록의 제목, 설명, 공개 상태와 같은 정보를 정의하는 playlist [https://developers.google.com/youtube/v3/docs/playlists?hl=ko] 리소스가 포함되어 있습니다. 이 예에서는 다음 요청 본문이 사용됩니다.
{
  "id": "PLAYLIST_ID",
  "snippet": {
    "title": "New playlist",
    "description": "New playlist description"
  },
  "status": {
    "privacyStatus": "public"
  }
}
다음 예에서는 재생목록을 팟캐스트 프로그램으로 표시합니다.
{
  "id": "PLAYLIST_ID",
  "snippet": {
    "title": "New playlist",
    "description": "New playlist description"
  },
  "status": {
    "podcastStatus": "enabled"
  }
}
참고: API 탐색기에서 이 쿼리를 테스트하는 경우 리소스 본문에 유효한 재생목록 ID를 대체해야 합니다. 재생목록 ID를 가져오려면 먼저 위에 표시된 재생목록 추가 [https://developers.google.com/youtube/v3/guides/implementation/playlists?hl=ko#playlists-insert] 요청을 실행하는 것이 좋습니다. API 응답에서 재생목록 ID를 추출하고 업데이트하려는 재생목록에 이 값을 사용합니다. 이 예에서 사용된 재생목록 제목과 설명은 위 예에서 사용된 것과 동일합니다.
재생목록 항목 추가
이 예에서는 재생목록에 항목을 추가합니다. 이 요청은 OAuth 2.0을 사용하여 승인해야 합니다. 이 예시에는 두 단계가 있습니다.
1단계: 적절한 재생목록 ID 가져오기
playlists.list [https://developers.google.com/youtube/v3/docs/playlists/list?hl=ko] 메서드를 호출하여 현재 인증된 사용자의 채널에 있는 재생목록을 가져옵니다. 위의 현재 사용자의 재생목록을 가져오는 [https://developers.google.com/youtube/v3/guides/implementation/playlists?hl=ko#playlists-retrieve-for-current-user] 샘플 요청은 이 요청을 보여줍니다. API를 호출하는 애플리케이션은 API 응답을 처리하여 각 재생목록의 ID를 키로 사용하여 재생목록 목록을 표시할 수 있습니다.
2단계: 재생목록에 동영상 추가하기
playlistItems.insert [https://developers.google.com/youtube/v3/docs/playlistItems/insert?hl=ko] 메서드를 호출하여 재생목록에 동영상을 추가합니다. 이 요청은 OAuth 2.0을 사용하여 승인해야 합니다. 요청 본문은 다음 값을 하나 이상 지정하는 playlistItem [https://developers.google.com/youtube/v3/docs/playlistItems?hl=ko] 리소스입니다.
snippet.playlistId [https://developers.google.com/youtube/v3/docs/playlistItems?hl=ko#snippet.playlistId]는 동영상을 추가할 재생목록을 식별합니다. 1단계에서 가져온 재생목록 ID입니다.
snippet.resourceId.kind [https://developers.google.com/youtube/v3/docs/playlistItems?hl=ko#snippet.resourceId.kind]에는 youtube#video 값이 포함되어 있습니다.
snippet.resourceId.videoId [https://developers.google.com/youtube/v3/docs/playlistItems?hl=ko#snippet.resourceId.videoId]는 재생목록에 추가하는 동영상을 식별합니다. 속성 값은 고유한 YouTube 동영상 ID입니다.
아래 API 요청은 재생목록에 동영상을 추가합니다. 요청 본문은 다음과 같습니다.
{
  "snippet": {
    "playlistId": "PLAYLIST_ID",
    "resourceId": {
      "kind": "youtube#video",
      "videoId": "VIDEO_ID"
    }
  }
}
API 탐색기에서 요청을 완료하려면 snippet.playlistId 및 snippet.resourceId.videoId 속성의 값을 설정해야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.playlistItems.insert?
        part=snippet [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.playlistItems.insert?part=snippet&resource=%257B%250A++%2522snippet%2522%253A+%250A++%257B%250A++++%2522playlistId%2522%253A+%2522PLAYLIST_ID%2522%252C%250A++++%2522resourceId%2522%253A%250A++++%257B++++++%2522kind%2522%253A+%2522youtube#video%2522%252C%250A++++++%2522videoId%2522%253A+%2522VIDEO_ID%2522%250A++++%257D%250A++%257D%250A%257D]
참고: playlistItems.insert [https://developers.google.com/youtube/v3/docs/playlistItems/insert?hl=ko#request_body] 메서드 문서에서는 재생목록 항목을 추가할 때 설정할 수 있는 다른 속성 값을 식별합니다.
재생목록 이미지 추가
v3 API의 playlistImages.insert [https://developers.google.com/youtube/v3/docs/playlistImages/insert?hl=ko] 메서드를 사용하여 맞춤 썸네일 이미지를 업로드하고 재생목록에 설정할 수 있습니다. 요청에서 playlistId [https://developers.google.com/youtube/v3/docs/playlistImages/insert?hl=ko#playlistId] 매개변수의 값은 썸네일이 사용될 재생목록을 식별합니다.
이 메서드의 요구사항인 미디어 파일 업로드 기능을 API 탐색기에서 지원하지 않으므로 이 쿼리는 API 탐색기를 사용하여 테스트할 수 없습니다.
재생목록 항목 업데이트
이 예에서는 재생목록의 첫 번째 항목이 되도록 재생목록 항목을 업데이트합니다. 이 요청은 OAuth 2.0을 사용하여 승인해야 합니다. 이 예에는 세 단계가 있습니다.
1단계: 적절한 재생목록 ID 가져오기
playlists.list [https://developers.google.com/youtube/v3/docs/playlists/list?hl=ko] 메서드를 호출하여 현재 인증된 사용자의 채널에 있는 재생목록을 가져옵니다. 위의 현재 사용자의 재생목록을 가져오는 [https://developers.google.com/youtube/v3/guides/implementation/playlists?hl=ko#playlists-retrieve-for-current-user] 샘플 요청은 이 요청을 보여줍니다. API를 호출하는 애플리케이션은 API 응답을 처리하여 각 재생목록의 ID를 키로 사용하여 재생목록 목록을 표시할 수 있습니다.
2단계: 선택한 재생목록의 항목 검색하기
playlistItems.list 메서드를 호출하여 선택한 재생목록의 동영상 목록을 가져옵니다. playlistId [https://developers.google.com/youtube/v3/docs/playlistItems/list?hl=ko#playlistId] 매개변수의 값을 1단계에서 가져온 재생목록 ID로 설정합니다.
API 응답의 각 리소스에는 해당 항목을 고유하게 식별하는 재생목록 항목 ID를 식별하는 id 속성이 포함되어 있습니다. 다음 단계에서 이 값을 사용하여 목록에서 항목을 삭제합니다.
3단계: 선택한 재생목록 항목 업데이트하기
playlistItems.update [https://developers.google.com/youtube/v3/docs/playlistItems/update?hl=ko] 메서드를 호출하여 재생목록에서 동영상의 위치를 변경합니다. part [https://developers.google.com/youtube/v3/docs/playlistItems/update?hl=ko#part] 매개변수 값을 snippet로 설정합니다. 요청 본문은 다음 값을 설정하는 playlistItem [https://developers.google.com/youtube/v3/docs/playlistItems?hl=ko] 리소스여야 합니다.
id 속성을 2단계에서 가져온 재생목록 항목 ID로 설정합니다.
snippet.playlistId 속성을 1단계에서 가져온 재생목록 ID로 설정합니다.
snippet.resourceId.kind 속성을 youtube#video로 설정합니다.
snippet.resourceId.videoId 속성을 재생목록에 포함된 동영상을 고유하게 식별하는 동영상 ID로 설정합니다.
snippet.position 속성을 0 또는 항목이 표시될 위치 (0 기반 색인 사용)로 설정합니다.
아래 API 요청은 재생목록 항목을 재생목록의 첫 번째 항목으로 업데이트합니다. 요청 본문은 다음과 같습니다.
{
  "id": "PLAYLIST_ITEM_ID",
  "snippet": {
    "playlistId": "PLAYLIST_ID",
    "resourceId": {
      "kind": "youtube#video",
      "videoId": "VIDEO_ID"
    },
    "position": 0
  }
}
API 탐색기에서 요청을 완료하려면 id, snippet.playlistId, snippet.resourceId.videoId 속성의 값을 설정해야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.playlistItems.update?

재생목록 이미지 업데이트
v3 API의 playlistImages.update [https://developers.google.com/youtube/v3/docs/playlistImages/update?hl=ko] 메서드를 사용하여 재생목록의 썸네일 이미지를 업데이트할 수 있습니다. 요청에서 playlistId [https://developers.google.com/youtube/v3/docs/playlistImages/update?hl=ko#playlistId] 매개변수의 값은 썸네일이 사용될 재생목록을 식별합니다.
이 메서드에 필요한 미디어 파일 업로드 기능을 API 탐색기에서 지원하지 않으므로 API 탐색기를 사용하여 이 쿼리를 테스트할 수 없습니다.
재생목록 항목 삭제
이 예에서는 재생목록에서 동영상을 삭제합니다. 이 요청은 OAuth 2.0을 사용하여 승인해야 합니다. 이 예에는 3단계가 있습니다. 처음 두 단계는 위의 재생목록 항목 업데이트 [https://developers.google.com/youtube/v3/guides/implementation/playlists?hl=ko#playlistItems-update] 예시와 동일합니다.
이 단계를 완료한 후 playlistItems.delete [https://developers.google.com/youtube/v3/docs/playlistItems/delete?hl=ko] 메서드를 호출하여 재생목록에서 동영상을 삭제합니다. 요청의 id [https://developers.google.com/youtube/v3/docs/playlistItems/delete?hl=ko#id] 매개변수를 삭제하려는 항목의 재생목록 항목 ID로 설정합니다. 이 요청은 OAuth 2.0을 사용하여 승인해야 합니다.
재생목록에서 동영상을 식별하는 데 사용되는 재생목록 항목 ID는 동영상을 고유하게 식별하는 YouTube 동영상 ID와 다릅니다. 재생목록 항목 ID는 동영상을 특정 재생목록의 항목으로 식별합니다.
API 탐색기에서 요청을 완료하려면 id 속성의 값을 설정해야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.playlistItems.delete?
        id=PLAYLIST_ITEM_ID [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.playlistItems.delete?id=PLAYLIST_ITEM_ID]
재생목록 이미지 삭제
이 예시에서는 재생목록 이미지를 삭제합니다. 이 예에는 두 단계가 있습니다.
1단계: 재생목록 검색하기
playlists.list [https://developers.google.com/youtube/v3/docs/playlists/list?hl=ko] 메서드를 호출하여 현재 인증된 사용자의 채널에 있는 재생목록을 가져옵니다. 위의 현재 사용자의 재생목록을 가져오는 [https://developers.google.com/youtube/v3/guides/implementation/playlists?hl=ko#playlists-retrieve-for-current-user] 샘플 요청은 이 요청을 보여줍니다. API를 호출하는 애플리케이션은 API 응답을 처리하여 각 재생목록의 ID를 키로 사용하여 재생목록 목록을 표시할 수 있습니다.
2단계: 재생목록 이미지 ID 가져오기
playlistImages.list [https://developers.google.com/youtube/v3/docs/playlistImages/list?hl=ko] 메서드를 호출하여 재생목록의 썸네일 이미지를 가져옵니다. 변경하려는 재생목록의 재생목록 ID를 사용합니다.
3단계: 재생목록 이미지 삭제하기
playlistImages.delete [https://developers.google.com/youtube/v3/docs/playlistImages/delete?hl=ko] 메서드를 호출하여 특정 재생목록 이미지를 삭제합니다. 요청에서 id 매개변수는 삭제되는 재생목록 이미지의 ID를 지정합니다. 이 메서드에는 유효한 OAuth 2.0 승인 토큰이 필요합니다. API 탐색기에서 이 쿼리를 테스트하는 경우 id 매개변수 값을 유효한 재생목록 ID로 대체해야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.playlistImages.delete?
        id=ID [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.playlistImages.delete?id=ID]
참고: API 탐색기에서 이 쿼리를 테스트하는 경우 id 매개변수 값을 유효한 재생목록 이미지 ID로 바꿔야 합니다.
