다음 예에서는 YouTube Data API (v3)를 사용하여 구독과 관련된 기능을 실행하는 방법을 보여줍니다.
채널의 구독 항목 검색
subscriptions.list [https://developers.google.com/youtube/v3/docs/subscriptions/list?hl=ko] 메서드를 호출하여 특정 채널의 구독을 가져옵니다. 채널을 식별하는 방법에는 두 가지가 있습니다.
현재 인증된 사용자의 구독을 검색하려면 mine [https://developers.google.com/youtube/v3/docs/subscriptions/list?hl=ko#mine] 매개변수의 값을 true로 설정합니다. mine 매개변수를 사용하는 요청은 OAuth 2.0을 사용하여 승인해야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.subscriptions.list?
        part=snippet,contentDetails
        &mine=true [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.subscriptions.list?part=snippet%252CcontentDetails&mine=true]
다른 채널의 구독을 검색하려면 channelId [https://developers.google.com/youtube/v3/docs/subscriptions/list?hl=ko#channelId] 매개변수 값을 해당 채널의 고유한 YouTube 채널 ID로 설정합니다. 아래 예에서는 YouTube의 TED 채널이 구독 중인 채널 목록을 가져옵니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.subscriptions.list?
        part=snippet,contentDetails
        &channelId=UCAuUUnT6oDeKwE6v1NGQxug [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.subscriptions.list?part=snippet%252CcontentDetails&channelId=UCAuUUnT6oDeKwE6v1NGQxug]
참고: 지정된 채널이 구독을 공개적으로 노출하지 않고 채널 소유자가 요청을 승인하지 않은 경우 API는 403 (Forbidden) HTTP 응답 코드를 반환합니다.
코드 샘플은 subscriptions.list [https://developers.google.com/youtube/v3/docs/subscriptions/list?hl=ko#usage] 메서드 문서를 참고하세요.
구독 추가하기
subscriptions.insert [https://developers.google.com/youtube/v3/docs/subscriptions/insert?hl=ko] 메서드를 호출하여 채널 구독을 추가합니다. 이 요청은 OAuth 2.0을 사용하여 승인해야 합니다. 요청 본문은 다음 값을 설정하는 subscription [https://developers.google.com/youtube/v3/docs/subscriptions?hl=ko] 리소스입니다.
snippet.resourceId.kind [https://developers.google.com/youtube/v3/docs/subscriptions?hl=ko#snippet.resourceId.kind]에는 youtube#channel 값이 포함되어 있습니다.
snippet.resourceId.channelId [https://developers.google.com/youtube/v3/docs/subscriptions?hl=ko#snippet.resourceId.channelId] 속성은 구독 중인 채널을 식별합니다. 속성 값은 고유한 YouTube 채널 ID입니다. 채널 ID는 channels.list [https://developers.google.com/youtube/v3/docs/channels/list?hl=ko] 메서드를 호출하거나 채널 검색 결과 [https://developers.google.com/youtube/v3/docs/search/list?hl=ko]를 검색하는 등 여러 가지 방법으로 가져올 수 있습니다.
아래 API 요청은 YouTube의 TED 채널을 구독합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.subscriptions.insert?
        part=snippet [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.subscriptions.insert?part=snippet&resource=%257B%250A++%2522snippet%2522%253A+%250A++%257B%250A++++%2522resourceId%2522%253A%250A++++%257B++++++%2522kind%2522%253A+%2522youtube#channel%2522%252C%250A++++++%2522channelId%2522%253A+%2522UCAuUUnT6oDeKwE6v1NGQxug%2522%250A++++%257D%250A++%257D%250A%257D]
요청 본문은 다음과 같습니다.
{
  "snippet": {
    "resourceId": {
      "kind": "youtube#channel",
      "videoId": "UCAuUUnT6oDeKwE6v1NGQxug"
    }
  }
}
코드 샘플은 subscriptions.insert [https://developers.google.com/youtube/v3/docs/subscriptions/insert?hl=ko#usage] 메서드 문서를 참고하세요.
구독 삭제
이 예시에서는 구독을 삭제합니다. 이 요청은 OAuth 2.0을 사용하여 승인해야 합니다. 이 예시에는 두 단계가 있습니다.
1단계: 인증된 사용자의 채널 구독 가져오기
subscriptions.list [https://developers.google.com/youtube/v3/docs/subscriptions/list?hl=ko] 메서드를 호출하여 정기 결제 목록을 가져옵니다. 위의 채널 구독을 검색하는 예시에서 이 요청을 실행하는 방법을 설명합니다.
API를 호출하는 애플리케이션은 API 응답을 처리하여 각 구독의 ID를 키로 사용하여 구독 목록을 표시할 수 있습니다. 응답에서 각 항목의 id 속성은 해당 정기 결제를 고유하게 식별하는 정기 결제 ID를 식별합니다. 다음 단계에서 이 값을 사용하여 목록에서 항목을 삭제합니다.
2단계: 구독 삭제하기
subscriptions.delete [https://developers.google.com/youtube/v3/docs/subscriptions/delete?hl=ko] 메서드를 호출하여 정기 결제를 삭제합니다. 요청의 id [https://developers.google.com/youtube/v3/docs/subscriptions/delete?hl=ko#id] 매개변수를 삭제하려는 구독의 구독 ID로 설정합니다. 이 요청은 OAuth 2.0을 사용하여 승인해야 합니다.
API 탐색기에서 요청을 완료하려면 id 속성의 값을 설정해야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.subscriptions.delete?
        id=SUBSCRIPTION_ID [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.subscriptions.delete?id=SUBSCRIPTION_ID]
코드 샘플은 subscriptions.delete [https://developers.google.com/youtube/v3/docs/subscriptions/delete?hl=ko#usage] 메서드 문서를 참고하세요.
승인된 사용자의 채널 구독자 목록 가져오기
현재 인증된 사용자의 채널을 구독하는 채널 목록을 가져오려면 subscriptions.list 메서드를 호출하고 mySubscribers [https://developers.google.com/youtube/v3/docs/subscriptions/list?hl=ko#mySubscribers] 매개변수의 값을 true로 설정합니다. 요청은 OAuth 2.0을 사용하여 승인되어야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.subscriptions.list?
        part=snippet,contentDetails
        &mySubscribers=true [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.subscriptions.list?part=snippet%252CcontentDetails&mySubscribers=true]
