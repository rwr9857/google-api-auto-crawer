YouTube Data API는 할당량 시스템 [https://developers.google.com/youtube/v3/determine_quota_cost?hl=ko]을 사용하여 개발자가 의도한 대로 서비스를 사용하고 서비스 품질을 부당하게 저하시키거나 다른 사용자의 액세스를 제한하는 API 클라이언트를 만들지 않도록 합니다.
YouTube Data API를 사용 설정한 프로젝트의 기본 할당량은 일일 10,000단위로, 이는 대부분의 API 사용자에게 충분한 양입니다. Google API 콘솔의 할당량 [https://console.cloud.google.com/iam-admin/quotas?hl=ko] 페이지에서 할당량 사용량을 확인할 수 있습니다.
기본 할당량을 초과하는 할당량을 추가로 요청하려면 먼저 프로젝트가 YouTube API 서비스 이용약관 [https://developers.google.com/youtube/terms/developer-policies?hl=ko]을 준수함을 보여주는 감사를 완료해야 합니다. 이를 통해 YouTube는 대규모 프로젝트의 의도된 사용 사례를 파악하고 YouTube의 API 서비스가 악용되지 않는 방식으로 사용되고 있는지 확인할 수 있습니다. YouTube 개발자 정책 준수에 관한 자세한 내용은 이 링크 [https://developers.google.com/youtube/terms/developer-policies-guide?hl=ko]를 참고하세요.
감사 시작
프로젝트 감사를 시작하려면 YouTube API 서비스 - 감사 및 할당량 확장 양식 [https://support.google.com/youtube/contact/yt_api_form?hl=ko]을 작성하여 제출하세요. YouTube API 서비스팀에서 최대한 빨리 연락을 드릴 것입니다.
추가 할당량 연장 요청
지난 12개월 이내에 API 규정 준수 감사를 완료했지만 추가 할당량 연장이 필요한 경우 감사를 받은 개발자 요청 양식 [https://support.google.com/youtube/contact/yt_api_audited_developer_requests_form?hl=ko]을 작성하여 제출하세요.
실패한 감사에 대한 이의신청
최근 API 규정 준수 감사에서 실패했으며 이 결정에 이의신청을 하려면 이의신청 양식 [https://support.google.com/youtube/contact/yt_api_appeals?hl=ko]을 작성하여 제출하세요.
정기 감사
Google은 품질을 보장하고 제품과 서비스를 개선하며 YouTube API 서비스 서비스 약관을 준수하는지 확인하기 위해 정기적으로 감사를 실시합니다. 정기 감사를 완료하라는 Google의 연락을 받은 경우 정기 감사 양식 [https://support.google.com/youtube/contact/yt_api_form?hl=ko]을 작성하세요.
지배권 변경
YouTube의 API 서비스에 액세스하는 API 프로젝트와 관련하여 지배권 변경 (예: 주식 매입 또는 매각, 합병 또는 기타 형태의 기업 거래)이 발생한 개발자 또는 개발자를 대신하여 API 클라이언트를 운영하는 당사자는 지배권 변경 양식 [https://support.google.com/youtube/contact/yt_api_change_of_control_form?hl=ko]을 작성하여 제출해야 합니다.
