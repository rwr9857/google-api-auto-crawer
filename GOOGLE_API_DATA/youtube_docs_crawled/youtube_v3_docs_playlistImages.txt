playlistImage 리소스는 재생목록과 연결된 썸네일 이미지를 식별합니다.
메서드
API는 playlistImages 리소스에 대해 다음 메서드를 지원합니다.
list [https://developers.google.com/youtube/v3/docs/playlistImages/list?hl=ko]
API 요청 매개변수와 일치하는 재생목록 이미지 컬렉션을 반환합니다. 지금 사용해보기 [https://developers.google.com/youtube/v3/docs/playlistImages/list?hl=ko#usage]
insert [https://developers.google.com/youtube/v3/docs/playlistImages/insert?hl=ko]
재생목록에 썸네일 이미지를 추가합니다. 지금 사용해보기 [https://developers.google.com/youtube/v3/docs/playlistImages/insert?hl=ko#usage]
업데이트 [https://developers.google.com/youtube/v3/docs/playlistImages/update?hl=ko]

속성
다음 표는 이 리소스에 표시되는 속성을 정의합니다.
속성
kind string
API 리소스의 유형을 식별합니다. 값은 youtube#playlistImage입니다.
id string
YouTube가 재생목록 이미지를 고유하게 식별하는 데 사용하는 ID입니다.
snippet object
snippet 객체에는 유형, 크기 등 재생목록 이미지에 관한 기본 세부정보가 포함됩니다.
snippet.playlistId string
이 이미지가 연결된 재생목록의 재생목록 ID입니다.
snippet.type string
이미지 유형입니다.
snippet.width string
이미지의 너비입니다.
snippet.height string
이미지의 높이입니다.
