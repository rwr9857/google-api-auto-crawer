매개변수
다음 표에는 이 쿼리가 지원하는 매개변수가 나와 있습니다. 나열된 모든 매개변수는 쿼리 매개변수입니다.
매개변수
필수 매개변수
part string
part 매개변수는 API 응답에 포함될 commentThread 리소스 속성 하나 이상의 쉼표로 구분된 목록을 지정합니다.

다음 목록에는 매개변수 값에 포함할 수 있는 part 이름이 포함되어 있습니다.
id
replies
snippet
필터(다음 매개변수 중 하나만 지정)
allThreadsRelatedToChannelId string
allThreadsRelatedToChannelId 매개변수는 지정된 채널과 연결된 모든 댓글 대화목록을 반환하도록 API에 지시합니다.
id string
id 매개변수는 검색해야 하는 리소스의 댓글 대화목록 ID를 쉼표로 구분된 목록으로 지정합니다.
videoId string
videoId 매개변수는 지정된 동영상 ID와 연결된 댓글 대화목록을 반환하도록 API에 지시합니다.
선택적 매개변수
maxResults unsigned integer
maxResults 매개변수는 결과 집합에 반환해야 하는 최대 항목 수를 지정합니다.

참고: 이 매개변수는 id [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko#id] 매개변수와 함께 사용하도록 지원되지 않습니다. 사용 가능한 값: 1~100 기본값은 20입니다.
moderationStatus string
이 매개변수는 제대로 승인된 요청 [https://developers.google.com/youtube/v3/guides/authentication?hl=ko]에서만 사용할 수 있습니다. 반환되는 댓글 대화목록을 특정 검토 상태로 제한하려면 이 매개변수를 설정합니다.

참고: 이 매개변수는 id [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko#id] 매개변수와 함께 사용하도록 지원되지 않습니다. 기본값은 published입니다.

허용되는 값은 다음과 같습니다.
heldForReview: 운영자의 검토 대기 중인 댓글 대화목록을 검색합니다. 최상위 댓글 또는 해당 댓글에 대한 하나 이상의 답글이 검토 대기 중인 경우 댓글 대화목록이 응답에 포함될 수 있습니다.
likelySpam: 스팸일 가능성이 높은 것으로 분류된 댓글 대화목록을 검색합니다. 최상위 댓글 또는 해당 댓글에 대한 하나 이상의 답글이 스팸으로 간주되는 경우 댓글 대화목록이 응답에 포함될 수 있습니다.
published – 게시된 댓글의 대화목록을 가져옵니다. 기본값입니다. 최상위 댓글이 게시된 경우 댓글 대화목록이 응답에 포함될 수 있습니다.
order string
order 매개변수는 API 응답이 댓글 대화목록을 나열해야 하는 순서를 지정합니다. 유효한 값은 다음과 같습니다.
time - 댓글 대화목록이 시간순으로 정렬됩니다. 이것이 기본 동작입니다.
relevance - 댓글 대화목록이 관련성에 따라 정렬됩니다.
참고: 이 매개변수는 id [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko#id] 매개변수와 함께 사용하도록 지원되지 않습니다.
pageToken string
pageToken 매개변수는 반환해야 하는 결과 집합의 특정 페이지를 식별합니다. API 응답에서 nextPageToken 속성은 검색할 수 있는 결과의 다음 페이지를 식별합니다.

참고: 이 매개변수는 id [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko#id] 매개변수와 함께 사용하도록 지원되지 않습니다.
searchTerms string
searchTerms 매개변수는 지정된 검색어가 포함된 댓글만 포함하도록 API 응답을 제한하도록 API에 지시합니다.

참고: 이 매개변수는 id [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko#id] 매개변수와 함께 사용하도록 지원되지 않습니다.
textFormat string
이 매개변수의 값을 html 또는 plainText로 설정하여 사용자가 작성한 댓글을 HTML 형식이나 일반 텍스트로 API에서 반환하도록 합니다. 기본값은 html입니다.

허용되는 값은 다음과 같습니다.
html – 주석을 HTML 형식으로 반환합니다. 기본값입니다.
plainText – 일반 텍스트 형식으로 주석을 반환합니다.

속성
다음 표는 이 리소스에 표시되는 속성을 정의합니다.
속성
kind string
API 리소스 유형을 식별합니다. 값은 youtube#commentThreadListResponse입니다.
etag etag
이 리소스의 Etag입니다.
nextPageToken string
pageToken 매개변수의 값으로 사용하여 결과 집합의 다음 페이지를 검색할 수 있는 토큰입니다.
pageInfo object
pageInfo 객체는 결과 집합의 페이지 정보를 요약합니다.
pageInfo.totalResults integer
결과 집합의 총 결과 수입니다.
pageInfo.resultsPerPage integer
API 응답에 포함된 결과 수입니다.
items[] list
요청 기준과 일치하는 댓글 대화목록의 목록입니다.
오류
다음 표에서는 이 메서드에 대한 호출에 대한 응답으로 API가 반환할 수 있는 오류 메시지를 확인합니다. 자세한 내용은 오류 메시지 [https://developers.google.com/youtube/v3/docs/errors?hl=ko] 문서를 참고하세요.
오류 유형 오류 세부정보 설명
badRequest (400) operationNotSupported id 필터는 Google+를 기반으로 하는 댓글에만 사용할 수 있습니다.
badRequest (400) processingFailure API 서버에서 요청을 처리하지 못했습니다. 일시적인 오류일 수 있지만 일반적으로 요청의 입력이 잘못되었음을 나타냅니다. 요청 본문에서 commentThread 리소스의 구조를 확인하여 유효한지 확인합니다.
forbidden (403) commentsDisabled videoId [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko#videoId] 매개변수로 식별된 동영상은 댓글을 사용 중지했습니다.
forbidden (403) forbidden 권한이 부족하여 요청된 댓글 대화목록 중 하나 이상을 가져올 수 없습니다. 요청이 제대로 인증되지 않았을 수 있습니다.
notFound (404) channelNotFound allThreadsRelatedToChannelId [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko#allThreadsRelatedToChannelId] 매개변수로 식별된 채널을 찾을 수 없습니다.
notFound (404) commentThreadNotFound 지정된 댓글 대화목록을 하나 이상 찾을 수 없습니다. 요청의 id [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko#id] 매개변수 값이 올바른지 확인하세요.
notFound (404) videoNotFound videoId [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko#videoId] 매개변수로 식별된 동영상을 찾을 수 없습니다.
