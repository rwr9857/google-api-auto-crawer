i18nLanguage 리소스는 YouTube 웹사이트에서 지원하는 애플리케이션 언어를 식별합니다. 애플리케이션 언어를 UI 언어라고도 합니다. YouTube 웹사이트의 경우 Google 계정 설정, 브라우저 언어 또는 IP 위치에 따라 애플리케이션 언어가 자동으로 선택될 수 있습니다. 사용자는 YouTube 사이트 바닥글에서 원하는 UI 언어를 직접 선택할 수도 있습니다.

각 i18nLanguage 리소스는 언어 코드와 이름을 식별합니다. 언어 코드는 videoCategories.list 및 guideCategories.list와 같은 API 메서드를 호출할 때 hl 매개변수의 값으로 사용할 수 있습니다.
메서드
API는 i18nLanguages 리소스에 다음 메서드를 지원합니다.
list [https://developers.google.com/youtube/v3/docs/i18nLanguages/list?hl=ko]
YouTube 웹사이트에서 지원하는 애플리케이션 언어 목록을 반환합니다. 지금 사용해보기 [https://developers.google.com/youtube/v3/docs/i18nLanguages/list?hl=ko#usage]
리소스 표현
다음 JSON 구조는 i18nLanguages 리소스의 형식을 보여줍니다.
{
  "kind [https://developers.google.com/youtube/v3/docs/i18nLanguages?hl=ko#kind]": "youtube#i18nLanguage",
  "etag [https://developers.google.com/youtube/v3/docs/i18nLanguages?hl=ko#etag]": etag,
  "id [https://developers.google.com/youtube/v3/docs/i18nLanguages?hl=ko#id]": string,
  "snippet [https://developers.google.com/youtube/v3/docs/i18nLanguages?hl=ko#snippet]": {
    "hl [https://developers.google.com/youtube/v3/docs/i18nLanguages?hl=ko#snippet.hl]": string,
    "name [https://developers.google.com/youtube/v3/docs/i18nLanguages?hl=ko#snippet.name]": string
  }
}
속성
다음 표는 이 리소스에 표시되는 속성을 정의합니다.
속성
kind string
API 리소스의 유형을 식별합니다. 값은 youtube#i18nLanguage입니다.
etag etag
이 리소스의 Etag입니다.
id string
YouTube가 i18n 언어를 고유하게 식별하는 데 사용하는 ID입니다.
snippet object
snippet 객체에는 언어 코드 및 이름과 같은 i18n 언어에 관한 기본 세부정보가 포함됩니다.
snippet.hl string
언어를 고유하게 식별하는 BCP-47 코드입니다.
snippet.name string
i18nLanguage.list 메서드의 hl [https://developers.google.com/youtube/v3/docs/i18nLanguages/list?hl=ko#hl] 매개변수를 사용하여 지정된 언어로 작성된 언어 이름입니다.
