guideCategories 리소스와 guideCategories.list 메서드는 2020년 9월 9일부터 모두 지원 중단되었습니다.
guideCategory 리소스는 채널의 콘텐츠 또는 채널의 인기도와 같은 기타 지표를 기반으로 YouTube에서 알고리즘을 통해 할당하는 카테고리를 식별합니다. 이 목록은 동영상 카테고리 [https://developers.google.com/youtube/v3/docs/videoCategories?hl=ko]와 유사하지만 동영상 카테고리는 동영상 업로더가 지정할 수 있고 채널 카테고리는 YouTube만 지정할 수 있다는 점이 다릅니다.
메서드
API는 guideCategories 리소스에 대해 다음 메서드를 지원합니다.
list [https://developers.google.com/youtube/v3/docs/guideCategories/list?hl=ko]
YouTube 채널과 연결할 수 있는 카테고리 목록을 반환합니다. 지금 사용해보기 [https://developers.google.com/youtube/v3/docs/guideCategories/list?hl=ko#try-it]

속성
다음 표는 이 리소스에 표시되는 속성을 정의합니다.
속성
kind string
API 리소스의 유형을 식별합니다. 값은 youtube#guideCategory입니다.
etag etag
이 리소스의 Etag입니다.
id string
YouTube에서 가이드 카테고리를 고유하게 식별하는 데 사용하는 ID입니다.
snippet object
snippet 객체에는 카테고리의 기본 세부정보(예: 제목)가 포함되어 있습니다.
snippet.channelId string
YouTube에서 가이드 카테고리를 게시하는 채널을 고유하게 식별하는 데 사용하는 ID입니다.
snippet.title string
카테고리의 제목입니다.
