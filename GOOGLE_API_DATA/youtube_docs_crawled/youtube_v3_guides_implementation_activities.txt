YouTube에서 채널 게시판 기능을 지원 중단합니다. 이에 따라 activities.insert [https://developers.google.com/youtube/v3/docs/activities/insert?hl=ko] 메서드는 지원 중단되고 activities.list [https://developers.google.com/youtube/v3/docs/activities/list?hl=ko] 메서드는 채널 게시판을 반환하지 않습니다. 이 변경사항은 2020년 5월 18일부터 API에 적용됩니다. 자세한 내용은 YouTube 고객센터 [https://support.google.com/youtube?p=channel-bulletins&hl=ko]를 참고하세요.
다음 예에서는 YouTube Data API (v3)를 사용하여 사용자 활동과 관련된 기능을 실행하는 방법을 보여줍니다.
채널 활동 목록 가져오기
특정 채널과 관련된 이벤트 목록을 검색하려면 다음 두 가지 방법 중 하나를 사용하여 activities.list [https://developers.google.com/youtube/v3/docs/activities/list?hl=ko] 메서드를 호출하여 채널을 식별합니다.
mine [https://developers.google.com/youtube/v3/docs/activities/list?hl=ko#mine] 매개변수 값을 true로 설정하여 현재 인증된 사용자의 활동 목록을 가져옵니다. 요청은 OAuth 2.0을 사용하여 승인되어야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.activities.list?
        part=snippet,contentDetails
        &mine=true [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.activities.list?part=snippet%252CcontentDetails&mine=true]
channelId [https://developers.google.com/youtube/v3/docs/activities/list?hl=ko#channelId] 매개변수를 활동 목록을 가져오는 채널을 고유하게 식별하는 YouTube 채널 ID로 설정합니다. 이 예에서는 channelId 매개변수를 UCK8sQmJBp8GCxrOtXWBpyEA로 설정합니다. 이 값은 Google의 공식 YouTube 채널을 식별하기도 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.activities.list?
        part=snippet,contentDetails
        &channelId=UCK8sQmJBp8GCxrOtXWBpyEA [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.activities.list?part=snippet%252CcontentDetails&channelId=UCK8sQmJBp8GCxrOtXWBpyEA]
구독 활동 목록 가져오기
구독 활동은 인증된 사용자가 구독하는 채널과 연결된 이벤트를 나타냅니다. 현재 인증된 사용자의 정기 결제 활동 목록을 가져오려면 activities.list [https://developers.google.com/youtube/v3/docs/activities/list?hl=ko] 메서드를 호출하고 home [https://developers.google.com/youtube/v3/docs/activities/list?hl=ko#home] 매개변수의 값을 true로 설정합니다. 요청은 OAuth 2.0을 사용하여 승인되어야 합니다.
https://developers.google.com/apis-explorer/#p/youtube/v3/youtube.activities.list?
        part=snippet,contentDetails
        &home=true [https://developers.google.com/apis-explorer/?hl=ko#p/youtube/v3/youtube.activities.list?part=snippet%252CcontentDetails&home=true]
