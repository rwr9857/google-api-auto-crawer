메서드 [https://developers.google.com/youtube/v3/docs/commentThreads?hl=ko#methods]
리소스 표현 [https://developers.google.com/youtube/v3/docs/commentThreads?hl=ko#resource-representation]
속성 [https://developers.google.com/youtube/v3/docs/commentThreads?hl=ko#properties]
commentThread 리소스에는 YouTube 댓글 대화목록에 대한 정보가 포함되어 있습니다. YouTube 댓글 대화목록은 해당 댓글에 대한 최상위 댓글과 답글(있는 경우)으로 구성됩니다. commentThread 리소스는 동영상의 댓글을 나타냅니다.

최상위 댓글과 답글 모두 실제로는 commentThread 리소스 내에 중첩된 comment [https://developers.google.com/youtube/v3/docs/comments?hl=ko] 리소스입니다. commentThread 리소스에 댓글에 대한 모든 답글을 포함할 필요는 없으며, 특정 댓글에 대한 모든 답글을 검색하려면 comments.list [https://developers.google.com/youtube/v3/docs/comments/list?hl=ko] 메서드를 사용해야 합니다. 일부 댓글에 답글이 없습니다.
메서드
API는 commentThreads 리소스에 다음 메서드를 지원합니다.
list [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko]
API 요청 매개변수와 일치하는 댓글 스레드 목록을 반환합니다. 지금 사용해보기 [https://developers.google.com/youtube/v3/docs/commentThreads/list?hl=ko#usage]
insert [https://developers.google.com/youtube/v3/docs/commentThreads/insert?hl=ko]
새 최상위 댓글을 작성합니다. 기존 댓글에 답글을 추가하려면 comments.insert [https://developers.google.com/youtube/v3/docs/comments/insert?hl=ko] 메서드를 대신 사용하세요. 지금 사용해 보세요 [https://developers.google.com/youtube/v3/docs/commentThreads/insert?hl=ko#usage].

속성
다음 표는 이 리소스에 표시되는 속성을 정의합니다.
속성
kind string
API 리소스의 유형을 식별합니다. 값은 youtube#commentThread입니다.
etag etag
이 리소스의 Etag입니다.
id string
YouTube가 댓글 대화목록을 고유하게 식별하는 데 사용하는 ID입니다.
snippet object
snippet 객체에는 댓글 대화목록에 관한 기본 세부정보가 포함됩니다. 또한 스레드의 최상위 주석(comment [https://developers.google.com/youtube/v3/docs/comments?hl=ko] 리소스)도 포함되어 있습니다.
snippet.channelId string
대화목록의 댓글과 연결된 YouTube 채널입니다. snippet.videoId 속성은 동영상을 식별합니다.
snippet.videoId string
댓글이 참조하는 동영상의 ID입니다.
snippet.topLevelComment object
대화목록의 최상위 댓글 속성 값은 comment [https://developers.google.com/youtube/v3/docs/comments?hl=ko#resource] 리소스입니다.
snippet.canReply boolean
이 설정은 현재 뷰어가 대화목록에 답장할 수 있는지 여부를 나타냅니다.
snippet.totalReplyCount unsigned integer
최상위 댓글에 대한 응답으로 제출된 총 답장 수입니다.
snippet.isPublic boolean
이 설정은 모든 댓글과 댓글 답글을 비롯한 대화목록을 모든 YouTube 사용자에게 표시할지 여부를 나타냅니다.
replies object
replies 객체는 댓글에 대한 답글 목록이 있는 경우 이를 포함하는 컨테이너입니다. replies.comments 속성은 주석 목록 자체를 나타냅니다.
replies.comments[] list
최상위 댓글에 대한 하나 이상의 답글 목록입니다. 목록의 각 항목은 comment [https://developers.google.com/youtube/v3/docs/comments?hl=ko#resource] 리소스입니다.

목록에는 제한된 수의 답글이 포함되며, 목록의 항목 수가 snippet.totalReplyCount [https://developers.google.com/youtube/v3/docs/commentThreads?hl=ko#snippet.totalReplyCount] 속성의 값과 같지 않는 한 답글 목록은 최상위 댓글에 사용할 수 있는 총 답글 수의 하위 집합입니다. 최상위 댓글에 대한 모든 답글을 검색하려면 comments.list [https://developers.google.com/youtube/v3/docs/comments/list?hl=ko] 메서드를 호출하고 parentId 요청 매개변수를 사용하여 답글을 검색하려는 댓글을 식별해야 합니다.
