이 페이지의 내용
독립형 코드 스니펫 [https://developers.google.com/youtube/v3/code_samples?hl=ko#standalone-code-snippets]
사용 사례 및 코드 샘플 [https://developers.google.com/youtube/v3/code_samples/code_snippets?hl=ko] 페이지에서 YouTube Data API 및 YouTube Live Streaming API의 일반적인 사용 사례를 살펴볼 수 있습니다.
이 페이지에서는 API 리소스와 메서드를 선택한 후 해당 메서드의 일반적인 사용 사례를 나열할 수 있습니다. 그런 다음 사용 사례를 클릭하여 해당 사용 사례의 샘플 매개변수 및 속성 값으로 API 탐색기 위젯을 채울 수 있습니다. 전체 화면 API 탐색기 위젯을 열어 Java, JavaScript, PHP, Python용으로 작동하는 코드 샘플을 확인할 수도 있습니다. API 탐색기에서 매개변수 또는 속성 값을 업데이트하면 코드 샘플도 변경사항을 반영하도록 업데이트됩니다.
다음 섹션에는 다른 언어로 제공되는 코드 샘플이 나와 있습니다.
독립형 코드 스니펫
아래 목록에는 Java, JavaScript, PHP, Python 이외의 언어로 YouTube Data API (v3)에 사용할 수 있는 코드 샘플이 나와 있습니다. 이전 섹션에서 언급한 대로 이러한 언어의 코드 샘플은 사용 사례 및 코드 샘플 [https://developers.google.com/youtube/v3/code_samples/code_snippets?hl=ko] 페이지에서 확인할 수 있습니다.
