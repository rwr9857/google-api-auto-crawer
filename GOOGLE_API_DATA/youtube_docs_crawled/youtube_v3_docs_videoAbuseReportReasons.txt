다음 표는 이 리소스에 표시되는 속성을 정의합니다.
속성
kind string
API 리소스의 유형을 식별합니다. 값은 youtube#videoAbuseReportReason입니다.
etag etag
이 리소스의 ETag입니다.
id string
YouTube에서 이유를 식별하는 데 사용하는 ID입니다. videos.reportAbuse 메서드를 호출할 때 애플리케이션은 이 값을 사용하여 reasonId [https://developers.google.com/youtube/v3/docs/videos/reportAbuse?hl=ko#reasonId] 속성을 설정해야 합니다.
snippet object
snippet 객체에는 이유에 관한 기본 세부정보가 포함됩니다.
snippet.label string
악용 신고 사유의 현지화된 라벨 텍스트입니다.
snippet.secondaryReasons[] list
해당 사유와 관련된 보조 사유 목록입니다(사용 가능한 경우). 0 이상일 수 있습니다.
snippet.secondaryReasons[].id string
YouTube에서 보조 이유를 식별하는 데 사용하는 ID입니다. videos.reportAbuse 메서드를 호출할 때 애플리케이션은 이 값을 사용하여 secondaryReasonId [https://developers.google.com/youtube/v3/docs/videos/reportAbuse?hl=ko#secondaryReasonId] 속성을 설정해야 합니다.
snippet.secondaryReasons[].label string
보조 이유의 현지화된 라벨 텍스트입니다.
