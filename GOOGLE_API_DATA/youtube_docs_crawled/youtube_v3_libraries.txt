클라이언트 라이브러리
bookmark_border
Google API 클라이언트 라이브러리는 여러 프로그래밍 언어로 사용할 수 있으며 YouTube API 구현을 단순화하는 데 크게 도움이 됩니다.
다음 표에서 첫 번째 열은 각 라이브러리의 개발 단계를 나타내며 (일부 라이브러리는 초기 단계임) 라이브러리의 문서로 연결됩니다. 두 번째 열은 각 라이브러리의 사용 가능한 샘플로 연결됩니다.

.NET용 Google API 클라이언트 라이브러리 [https://developers.google.com/api-client-library/dotnet/get_started?hl=ko] .NET 샘플 [https://developers.google.com/api-client-library/dotnet/apis?hl=ko]
