리소스 표현
메서드
리소스 설정은 사용자의 시간대와 같이 사용자가 Calendar UI에서 변경할 수 있는 설정을 나타냅니다. list [https://developers.google.com/workspace/calendar/api/v3/reference/settings/list?hl=ko] 및 get [https://developers.google.com/workspace/calendar/api/v3/reference/settings/get?hl=ko] 메서드를 통해 가져올 수 있습니다.설정에 기본값이 있으면 반환되지 않을 수 있습니다.

지원되는 설정 목록은 다음과 같습니다.
설정 ID 설명 허용되는 값 기본값
autoAddHangouts 모든 일정에 화상 회의 (Meet 또는 부가기능)를 자동으로 추가할지 여부입니다. 이 설정에 따라 논리를 처리하는 것은 클라이언트의 책임이므로 contactsDataVersion이 0보다 크면 서버에서 이 설정을 무시합니다. 읽기 전용입니다. 'true', 'false' 'false'
dateFieldOrder 날짜를 표시할 때 일 (D), 월 (M), 연도 (Y)의 순서를 어떻게 해야 하나요? 'MDY', 'DMY', 'YMD' 'MDY'
defaultEventLength 명시적 기간 없이 생성된 일정의 기본 길이 (분)입니다. 양수 '60'
format24HourTime 24시간 형식으로 시간을 표시할지 여부입니다. “true”, “false” 'false'
hideInvitations 사용자가 초대되었지만 응답 등 조치를 취하지 않은 이벤트를 숨길지 여부입니다. 'true', 'false' '거짓'
hideWeekends 한 주를 표시할 때 주말을 숨겨야 하는지 여부입니다. “true”, “false” '거짓'
locale 사용자의 언어입니다. "in", "ca","cs", "da", "de", "en_GB", "en", "es", "es_419", "tl", "fr", "hr", "it", "lv", "lt", "hu", "nl", "no", "pl", "pt_BR", "pt_PT", "ro", "sk", "sl", "fi", "sv", "tr", "vi", "el", "ru", "sr", "uk", "bg", "iw", "ar", "fa", "hi", "th", "zh_TW", "zh_CN", "ja", "ko" 'en'
remindOnRespondedEventsOnly 사용자의 응답 상태가 '예' 및 '미정'인 이벤트에 대해서만 일정 리마인더를 전송해야 하는지 여부입니다. 'true', 'false' 'false'
showDeclinedEvents 사용자가 '아니요'라고 응답한 일정이 사용자 캘린더에 표시되어야 하는지 여부입니다. 'true', 'false' “true”
시간대 사용자 시간대의 ID입니다. http://www.iana.org/time-zones [http://www.iana.org/time-zones]를 참조하세요. '기타/GMT'
useKeyboardShortcuts 단축키 사용 설정 여부 “true”, “false” “true”
weekStart 주가 일요일(0), 월요일(1) 또는 토요일(6)에 시작해야 하는지 여부입니다. '0', '1', '6' “0”
이 리소스에 대한 메소드 [https://developers.google.com/workspace/calendar/api/v3/reference/settings?hl=ko#methods] 목록은 이 페이지의 하단에 나와 있습니다.
리소스 표현
{
  "kind": "calendar#setting",
  "etag": etag,
  "id": string,
  "value": string
}
속성 이름 값 설명 참고
etag etag 리소스의 ETag입니다.
id string 사용자 설정의 ID입니다.
kind string 리소스의 유형('calendar#setting')입니다.
value string 사용자 설정의 값입니다. 값의 형식은 설정의 ID에 따라 다릅니다. 항상 길이가 최대 1,024자(영문 기준)인 UTF-8 문자열이어야 합니다.

get [https://developers.google.com/workspace/calendar/api/v3/reference/settings/get?hl=ko]
단일 사용자 설정을 반환합니다.
list [https://developers.google.com/workspace/calendar/api/v3/reference/settings/list?hl=ko]

시청 [https://developers.google.com/workspace/calendar/api/v3/reference/settings/watch?hl=ko]
설정 리소스의 변경사항을 확인합니다.
