리소스 표현
메서드
사용자의 캘린더 목록에 있는 캘린더 컬렉션입니다. 캘린더 및 CalendarList [https://developers.google.com/workspace/calendar/api/concepts/events-calendars?hl=ko#calendar_and_calendar_list]도 참고하세요.
이 리소스에 대한 메소드 [https://developers.google.com/workspace/calendar/api/v3/reference/calendarList?hl=ko#methods] 목록은 이 페이지의 하단에 나와 있습니다.

속성 이름 값 설명 참고
accessRole string 인증된 사용자가 캘린더에서 갖는 유효한 액세스 역할입니다. 읽기 전용입니다. 가능한 값은 다음과 같습니다.
'freeBusyReader' - 한가함/바쁨 정보에 대한 읽기 액세스 권한을 제공합니다.
'reader' - 캘린더에 대한 읽기 액세스 권한을 제공합니다. 비공개 이벤트는 리더 액세스 권한이 있는 사용자에게 표시되지만 이벤트 세부정보는 숨겨집니다.
'writer' - 캘린더에 대한 읽기 및 쓰기 액세스 권한을 제공합니다. 비공개 이벤트는 작성자 액세스 권한이 있는 사용자에게 표시되고 이벤트 세부정보가 표시됩니다.
'owner' - 캘린더의 소유권을 제공합니다. 이 역할은 작성자 역할의 모든 권한과 함께 ACL을 확인하고 조작할 수 있는 추가 권한을 갖습니다.
backgroundColor string 16진수 형식 "#0088aa"로 된 캘린더의 기본 색상입니다. 이 속성은 색인 기반 colorId 속성을 대체합니다. 이 속성을 설정하거나 변경하려면 insert [https://developers.google.com/calendar/v3/reference/calendarList/insert?hl=ko], update [https://developers.google.com/calendar/v3/reference/calendarList/update?hl=ko], patch [https://developers.google.com/calendar/v3/reference/calendarList/patch?hl=ko] 메서드의 매개변수에서 colorRgbFormat=true를 지정해야 합니다. 선택사항입니다. 쓰기 가능
colorId string 캘린더의 색상입니다. 색상 정의의 calendar 섹션에 있는 항목을 참조하는 ID입니다 (색상 엔드포인트 [https://developers.google.com/calendar/v3/reference/colors?hl=ko] 참고). 이 속성은 backgroundColor 및 foregroundColor 속성으로 대체되며 이러한 속성을 사용할 때 무시해도 됩니다. 선택사항입니다. 쓰기 가능
conferenceProperties nested object 이 캘린더의 회의 속성(예: 허용되는 회의 유형)입니다.
conferenceProperties.allowedConferenceSolutionTypes[] list 이 캘린더에서 지원되는 회의 솔루션 유형입니다.
사용할 수 있는 값은 다음과 같습니다.
"eventHangout"
"eventNamedHangout"
"hangoutsMeet"
선택사항.
defaultReminders[] list 인증된 사용자가 이 캘린더에 대해 가지고 있는 기본 알림입니다. 쓰기 가능
defaultReminders[].method string 이 알림에서 사용하는 방법입니다. 가능한 값은 다음과 같습니다.
'email' - 알림은 이메일을 통해 전송됩니다.
'popup' - 알림은 UI 팝업을 통해 전송됩니다.
알림을 추가할 때 필요합니다.
쓰기 가능
defaultReminders[].minutes integer 일정이 시작되기 전까지 알림이 트리거되어야 하는 시간(분)입니다. 유효한 값은 0에서 40320 (분 기준 4주) 사이입니다.
알림을 추가할 때 필요합니다.
쓰기 가능
deleted boolean 이 캘린더 목록 항목이 캘린더 목록에서 삭제되었는지 여부입니다. 읽기 전용입니다. 선택사항입니다. 기본값은 False입니다.
description string 캘린더에 대한 설명입니다. 선택사항입니다. 읽기 전용입니다.
etag etag 리소스의 eETag입니다.
foregroundColor string 16진수 형식 '#ffffff'으로 된 캘린더의 전경색입니다. 이 속성은 색인 기반 colorId 속성을 대체합니다. 이 속성을 설정하거나 변경하려면 insert [https://developers.google.com/calendar/v3/reference/calendarList/insert?hl=ko], update [https://developers.google.com/calendar/v3/reference/calendarList/update?hl=ko], patch [https://developers.google.com/calendar/v3/reference/calendarList/patch?hl=ko] 메서드의 매개변수에서 colorRgbFormat=true를 지정해야 합니다. 선택사항입니다. 쓰기 가능
hidden boolean 캘린더가 목록에서 숨김 처리되었는지 여부입니다. 선택사항입니다. 이 속성은 캘린더가 숨겨져 있는 경우에만 반환되며 이 경우 값은 true입니다. 쓰기 가능
id string 캘린더의 식별자입니다.
kind string 리소스의 유형('calendar#calendarListEntry')입니다.
location string 자유 형식 텍스트로 된 캘린더의 지리적 위치입니다. 선택사항입니다. 읽기 전용입니다.
notificationSettings object 인증된 사용자가 이 캘린더에 대해 수신하는 알림입니다. 쓰기 가능
notificationSettings.notifications[] list 이 캘린더에 설정된 알림 목록입니다.
notificationSettings.notifications[].method string 알림을 전달하는 데 사용되는 메서드입니다. 가능한 값은 다음과 같습니다.
'email' - 알림은 이메일을 통해 전송됩니다.
알림을 추가할 때 필요합니다.
쓰기 가능
notificationSettings.notifications[].type string 알림의 유형입니다. 가능한 값은 다음과 같습니다.
'eventCreation' - 캘린더에 새 일정이 추가될 때 전송되는 알림입니다.
'eventChange' - 이벤트가 변경되면 전송되는 알림입니다.
'eventCancellation' - 이벤트가 취소될 때 전송되는 알림.
'eventResponse' - 참석자가 일정 초대에 응답할 때 전송되는 알림입니다.
'agenda' - 오늘의 일정목록 (아침에 전송됨)
알림을 추가할 때 필요합니다.
쓰기 가능
primary boolean 캘린더가 인증된 사용자의 기본 캘린더인지 여부입니다. 읽기 전용입니다. 선택사항입니다. 기본값은 False입니다.
selected boolean 캘린더 콘텐츠가 캘린더 UI에 표시되는지 여부입니다. 선택사항입니다. 기본값은 False입니다. 쓰기 가능
summary string 캘린더 제목입니다. 읽기 전용입니다.
summaryOverride string 인증된 사용자가 이 캘린더에 설정한 요약입니다. 선택사항입니다. 쓰기 가능
timeZone string 캘린더의 시간대입니다. 선택사항입니다. 읽기 전용입니다.
메서드
삭제 [https://developers.google.com/workspace/calendar/api/v3/reference/calendarList/delete?hl=ko]
사용자의 캘린더 목록에서 캘린더를 삭제합니다.
get [https://developers.google.com/workspace/calendar/api/v3/reference/calendarList/get?hl=ko]
사용자의 캘린더 목록에서 캘린더를 반환합니다.
insert [https://developers.google.com/workspace/calendar/api/v3/reference/calendarList/insert?hl=ko]
기존 캘린더를 사용자의 캘린더 목록에 삽입합니다.
list [https://developers.google.com/workspace/calendar/api/v3/reference/calendarList/list?hl=ko]
사용자의 캘린더 목록에 있는 캘린더를 반환합니다.
patch [https://developers.google.com/workspace/calendar/api/v3/reference/calendarList/patch?hl=ko]
사용자의 캘린더 목록에 있는 기존 캘린더를 업데이트합니다. 이 메소드는 패치 의미 체계를 지원합니다. 각 패치 요청은 세 개의 할당량 단위를 사용합니다. get 다음에 update를 사용하는 것이 좋습니다. 지정한 필드 값이 기존 값을 대체합니다. 요청에서 지정하지 않은 필드는 변경되지 않습니다. 배열 필드(지정된 경우)는 기존 배열을 덮어씁니다. 이전 배열 요소가 모두 삭제됩니다.
업데이트 [https://developers.google.com/workspace/calendar/api/v3/reference/calendarList/update?hl=ko]
사용자의 캘린더 목록에 있는 기존 캘린더를 업데이트합니다.
시계 [https://developers.google.com/workspace/calendar/api/v3/reference/calendarList/watch?hl=ko]
CalendarList 리소스의 변경사항을 확인합니다.
